## 本文件是什么（核心定义）

- **交付物 ≠ 字段 schema**。交付物是一份"**玩法库 + 元素库 + 红线**"，是 AI 生产这种内容时可调用的**表达资源包**。
- **模板 ≠ 文字填空模板**。每张玩法卡里的「30 秒分镜参考」就是模板——**拍摄者照着能拍出来的执行卡片**。
- **AI 生产时不是"填字段"**，而是"**从玩法库里选一张卡片并展开**"。

> 定义来自用户 2026-04-13 裁决。详见 [_index.md](_index.md) §0。

---

## 与 `event_documentary` 的边界（来自 event_documentary 文件 G4）

本 ContentType 和 `event_documentary`（事件纪实）最容易混淆，因为两者都涉及"后台"。边界如下：

| 维度 | event_documentary | behind_the_scenes（本文件） |
|---|---|---|
| **后台的定位** | 后台是"**事件的一部分**"——为开场/主角/成品服务 | **后台本身就是"内容本体"**——不需要前台就能独立成立 |
| **叙事重心** | 一场具体活动的时间和空间 | **品牌日常经营的"**看不见的完成成本**" |
| **时间特点** | 事件的前-中-后三阶段 | **任意时刻**（拍摄 / 上新 / 补货 / 经营） |
| **最核心反套路** | 反"圆满落幕" | **反"只展示光鲜"** |

**生活化例子：** 同样拍"店员熨衣服"——
- 如果熨衣服是为了开业当天——归 `event_documentary`
- 如果熨衣服是为了呈现"日常经营里的劳动"——归 `behind_the_scenes`

## A. 北极星

**观众看完心里产生"原来你们也会手忙脚乱、也会临时救火、也会笑场，但正因为这样我更喜欢你们"的亲近感——"这家公司愿意把不完美给我看"。**

> **三份原话备查：**
> - GPT5.4：原来你们也会手忙脚乱、也会临时救火、也会笑场，但正因为这样我更喜欢你们
> - 深度研究：这家公司愿意把不完美给我看——从而产生亲近与信任
> - compass：从"拍摄花絮"到"治愈系 ASMR 补货日"——**幕后即内容**，不是正片附属品

## B. 元素库（原子层）

### B1 视觉元素
- 边角料物件（别针 / 衣架 / 透明胶 / 蒸汽熨斗 / mark 点 / 试衣间门帘）
- 手部动作特写（熨 / 别 / 捡 / 搬 / 装）
- 失败瞬间（衣服没熨平 / 模特迟到 / 拍到一半下雨）
- 非 glamorous 劳动（搬箱子、捡地上胶带、返袋、清点）
- 分屏构图（左边优雅 / 右边手忙脚乱）
- 留白黑屏（"成片明天发"）
- 低机位 / 物件视角
- 电影标题卡风格（韦斯·安德森 / 法国新浪潮）

### B2 听觉元素
- 卷帘门 / 门铃 / 衣架 / 扫码 / 剪吊牌
- 熨烫蒸汽 / 纸袋摩擦 / 试衣间门
- 拉链 / 关灯声
- 现场对话碎片（"等一下这里太平了"）
- 模特/同事的半句话
- 极弱 BGM 或完全无 BGM
- 10 个声音组成一条幕后

### B3 人物元素
- 熨衣服的人 / 捡胶带的人 / 搬箱子的人 / 装袋的人
- 幕后人群像（库管 / 熨烫 / 搭配 / 客服 / 收银 / 店长 各 6 秒）
- 同事临时救场
- 创作者本人边做边拍
- 一人分饰（"台前体面" vs "幕后笨拙"）

### B4 场景元素
- 店面后仓 / 仓库 / 发货区
- 拍摄现场（反光板 / 补光 / 试衣架）
- 办公室 / 样品间 / 样衣架前
- 试衣间门帘内外
- 清晨补货 / 夜里关店

### B5 情绪元素
- 手忙脚乱但不慌
- 笑场但不演
- 自嘲但不羞辱
- 收尾的疲惫
- 救火后的默契
- 留白带来的期待

### B6 文字元素
- 极简字幕（一句"等一下这里太平了"）
- 一句标准（每个幕后人物一句话）
- 黑屏字幕（"成片明天发"）
- 电影标题卡（韦斯·安德森黄色字体）
- 反鸡汤

### B7 时间元素
- 同一秒的两面（分屏同时看）
- 准备动作链（挑衣 → 熨烫 → 搬架子 → 找光）
- 一条内容的诞生（选衣 → 找光 → 试拍 → 失败 → 改 → 成片）
- 延时（补货日）

### B8 反差元素
- 台前体面 vs 幕后笨拙
- 拍摄完美 vs 现实混乱
- 结果很顺 vs 过程很崩
- 模特优雅转身 vs 有人蹲地拉裙摆

### B9 隐喻元素
- 幕后物件 = 劳动的见证
- 声音 = 店活着的证据
- 笨拙 = 品牌的人味
- 留白 = 观众的想象空间
- 自嘲 = 品牌自信的反向证明

## D. 服装行业垂直机会

`behind_the_scenes` 在服装行业最大的机会，是把"**美**"的**反面**拍出来：**蒸汽、皱褶、别针、搬箱、返袋、选图争执、临时救场**。服装品牌最容易犯的错，是**只愿意给观众看"优雅结果"**——但真正让人爱上的往往是："**你们原来也会手忙脚乱，却还是把事情做成了**"。

### 三个未被开发的角度

1. **"边角物件叙事"（C3-3 边角料主视角）** —— 服装行业的幕后物件**太丰富了**，其他行业没有这么丰富的"劳动痕迹物件"
2. **"店的声纹"（C3-4 10 个声音 + C3-5 补货日 ASMR）** —— 服装行业声音密度高且独特，**声纹可以变成品牌识别资产**
3. **"前台 / 后台并置"（C3-2 同一秒的两面）** —— 视觉反差极强，观众心理接受度高

### 服装行业 behind_the_scenes 方向（v0.2 M4 无痕处理：删去全部具体品牌 / 账号 / 数据 / 仅保留方向描述）
- **搞笑梗 + 员工文化方向** —— 把"幕后"做成品牌人格 / 用幽默打破"品牌正经感"
- **产品发布娱乐节目化方向** —— 用"等候名单 / 倒计时 / 预告片"做幕后包装
- **TikTok 短片式时尚电影方向** —— 把每条幕后做成 30 秒电影
- **品牌标志性幽默方向** —— 用"产品经历各种考验"作为长期幽默叙事
- **每条都是短片方向** —— "每条幕后都精心打光、有意图性、永恒感"
- **从零开设实体店全过程方向** —— 长期连载 + 翻车记录 / 把开店本身做成内容
- **高频更新穿搭方向** —— 一周三至四更的稳定内容产出节奏
- **每周拍摄日反复迭代方向** —— 把"内容生产本身"作为长期叙事素材

### 行业趋势支撑
- **TikTok 2026 "authenticity > polished"** 对 BTS 尤其适用
- **2025-2026 趋势关键词** ：黄油黄色衬线字体叠加 + 复古电影风（韦斯·安德森 / 法国新浪潮风格）
- "**真实性重要**" 成为普遍诉求——幕后内容是最省成本的信任资产

## 与前 5 份（企业叙事类全员）的差异点

| 维度 | founder_ip | role_work_vlog | event_documentary | product_journey | process_trace | behind_the_scenes（本文件） |

| **叙事重心** | 老板这个人 | 岗位工作过程 | 事件的时间空间 | 产品的判断历程 | 生产工艺的可验证 | **日常经营的"看不见的完成成本"** |
| **北极星** | 判断+代价+信任预支 | 尊重+体感信任 | 临场感+活着 | 不是突然出现的 | 相信你没骗我 | **愿意把不完美给我看** |
| **C3 突破性玩法数量** | 8 张 | 5 张 | 7 张 | 8 张 | 8 张 | **8 张** |
| **共享引擎映射** | 2 处 | 2 处 | 3 处 | 2 处 | 2 处 | **3 处（#2 认知反转 / #11 ASMR / #12 延时）** |
| **长期积累型玩法** | 无 | 无 | 1 张 | 2 张 | 2 张 | **2 张（C3-7 共创连续剧 / C3-8 自嘲日志）** |
| **最核心的反套路** | 反"老板精英感" | 反"可爱员工人设" | 反"圆满落幕" | 反"从草图到成品的爽片" | 反"走进工厂的宣传片" | **反"只展示光鲜"** |
| **独占价值** | 创始人 IP | 岗位独占知识 | 现场临场感 | 差一点做成别的样子 | 一个难以作假的瞬间 | **边角料的劳动痕迹** |

**一句话总结企业叙事类 6 个的内在逻辑：**
- `founder_ip` = 讲"这个人"
- `role_work_vlog` = 讲"这群人的手"
- `event_documentary` = 讲"那一次"
- `product_journey` = 讲"这件衣服怎么来"
- `process_trace` = 讲"这件衣服为什么值"
- `behind_the_scenes` = 讲"这一切是怎么被撑起来的"——**所有前 5 个的"地基"**
