## 本文件是什么（核心定义，防止脱离索引被误读）

- **交付物 ≠ 字段 schema**。交付物是一份"**玩法库 + 元素库 + 红线**"，是 AI 生产这种内容时可调用的**表达资源包**。
- **模板 ≠ 文字填空模板**。每张玩法卡里的「30 秒分镜参考」就是模板——**拍摄者照着能拍出来的执行卡片**，含画面 + 声音 + 动作 + 节奏。
- **AI 生产时不是"填字段"**，而是"**从玩法库里选一张卡片并展开**"。

> 定义来自用户 2026-04-13 裁决。详见 [_index.md](_index.md) §0。

## ⚠️ 本文件的特殊风险：为什么 emotion_expression 需要双层真实性防御

### 1. 这是整个 Q2 里最容易 AI 翻车的 ContentType

`founder_ip` 容易翻车是"精英感"—— 但观众还愿意给它一次机会。
`emotion_expression` 一旦翻车是"AI 腔" —— 观众**当场划走** + **反感度远高于 founder_ip**。

原料三份报告用了相近的警告：
- 通用类.md §5 红线："情绪全靠哭腔和悲伤音乐推 / 金句太满像朋友圈文案库"
- 通用类compass.md §5："46.28% 的年轻消费者动机是'取悦自我'……但**加速暴力把深度情感体验扁平化为转瞬即逝的神经冲动**"
- 通用类deep-research-report.md §emotion_expression 红线："**用 AI 腔长诗配 AI 配音：观众会立刻识别为'情绪模板'**"

### 2. 双层真实性防御系统（用户 2026-04-14 裁决 α + β 组合）

**α 层 ——「9 种 AI 腔禁用清单」作为 C1 档扩展红线**：告诉生产者"这 9 种长成这样的都别碰" —— 负面样本清单（详见 §E 红线段）

**β 层 ——「真实感三问自检」作为 C2/C3 每张玩法卡的必答项**：告诉生产者"做对的要同时满足 3 个锚点" —— 正面锚定

**三问内容**（每张进阶 / 突破性玩法卡必须在 § 实拍可行性 上方新增「真实感自检」字段）：

1. **画面锚点**：这条内容里的画面 / 声音 / 地点 / 时间是**当下真实发生**的吗？（如果画面是摆拍的 / 是从素材库调的 / 是从"情绪模板"生成的 → 不是 emotion_expression）
2. **情感锚点**：情绪是你**此刻的情绪**，还是"构思出来的情绪剧本"？（如果是为了拍内容而装出来的情绪 → 不是 emotion_expression，归到 `humor_content` 的自我调侃或 `lifestyle_expression` 的生活方式）
3. **外部锚点**：有没有**外部可验证的点** —— 真实日期 / 真实地点 / 真实他人在场 / 真实天气 / 真实物件磨损痕迹？（如果答案是"都没有"，说明这条内容完全活在内部臆想里 → 归到艺术向 `lifestyle_expression`）

**三问任一答'不' → 这条不应该作为 emotion_expression 发布**。可以降级到 `humor_content` / `lifestyle_expression` / `daily_fragment`。

### 3. 与 batch 1 的 `lifestyle_expression` / `humor_content` 的硬边界

| ContentType | 关键区分 |
|---|---|
| `lifestyle_expression` | 展示"**我的生活方式**"—— 观点 / 审美 / 价值观驱动 |
| `humor_content` | 通过"**幽默**"消解情绪 —— 笑是出口 |
| `daily_fragment` | 真实日常碎片，但**情绪密度低** —— 是"生活"不是"情绪"|
| **`emotion_expression`** | 叙事重心是"**某个情绪瞬间被看见 / 被接住 / 被共鸣**" —— 情绪密度高 + 真实锚点齐 |

## A. 北极星

**观众看完后，心里不是"这句话真会写"，而是"**这感觉你居然也知道**" —— 不是一个结论被砸下来，而是一个没说出口的感受被轻轻接住。我被看见了，我愿意把自己的情绪也放出来（点赞 / 评论 / 转发 / 收藏），我对这个人 / 这个品牌产生了更深层的情感连接。**

> **三份原话备查：**
> - 通用类.md：这感觉你居然也知道
> - 通用类compass.md：情绪资产（Emotional assets）—— 情绪共鸣内容矩阵是品牌留存与溢价的底层设计
> - 通用类deep-research-report.md：我被看见了 / 我愿意把自己的情绪放出来 / 我对这个品牌产生更深层的情感连接

## B. 元素库（原子层）

### B1 视觉元素
- **物件作为情绪载体**（未拆的快递 / 没脱的工牌 / 歪着的鞋 / 椅背上的外套 / 冷掉的咖啡）
- 空镜 / 留白镜头（桌角 / 窗外 / 天花板 / 地板光斑）
- 手部特写（脱外套 / 抱在怀里 / 拨头发 / 揉眼睛 / 放下包）
- 真实生活场景（玄关 / 厨房 / 地铁口 / 楼梯间 / 沙发角 / 门店后台）
- 中性表情镜头（不哭不笑 / 发呆 / 低头 / 看远处）
- 第一人称视角（镜头是主角的眼睛）
- 色温渐变（暖光 → 冷光 / 或反之）
- 慢动作本体（不是剪辑慢，是**生活动作本身慢** —— 把包放下用 5 秒）
- 光斑 / 逆光 / 半阴影（不要全亮 / 不要打光精致）
- **避免**：怼脸痛哭 / 黑白滤镜 / 戏剧化特写 / 金句 PPT

### B2 听觉元素
- 环境音作为情绪主角（电饭煲跳闸 / 楼下车声 / 门锁声 / 钥匙串 / 雨声 / 空调声）
- 微信提示音但不点开
- 布料声（脱外套 / 衣架碰撞 / 拉链）
- 呼吸声 / 脚步声 / 椅子吱呀声
- **极弱 BGM 或完全无 BGM**（emo 配乐是 C1 陷阱的死穴）
- 低声独白（如果有）—— 不是口播，是像写给自己的备注
- 半句话 / 停顿 / 欲言又止
- **避开**：伤感钢琴 / 悲怆弦乐 / 热门 emo 混音 / AI 配音合成声

### B3 人物元素
- 单人（默认 / 常以背影 / 侧影 / 只露手出现）
- 双人关系动作（妈妈 / 同事 / 伴侣 / 朋友，一个很小的举动代替对白）
- 不露脸（只拍物件和空间）
- 关系碎片（只拍动作，不拍脸）
- **避免**：怼脸独白 / 哭腔特写 / 戏剧化表情

### B4 场景元素
- 家中（玄关 / 沙发 / 厨房 / 卧室 / 椅子旁）
- 门店后台 / 试衣间 / 收银台 / 门口
- 通勤路（地铁站 / 楼梯间 / 人行道 / 红绿灯）
- 窗边 / 门前 / 楼道（"半室内半室外"的过渡场景）
- 店铺打烊后 / 家里深夜（情绪沉淀的时段）

### B5 情绪元素
- 克制（不爆发 / 不大喊 / 不大哭）
- 延迟（动作比平时慢 / 反应比平时晚）
- 轻度倦意（不是"累死了"而是"一直悬着没落地"）
- 被照顾 / 被看见 / 被接住的瞬间
- 羞怯 / 欲言又止 / 没说出口
- 允许软弱（不是宣告坚强）
- **避开**：戏剧化崩溃 / 励志反转 / 自我感动 / 道德绑架式共情

### B6 文字元素
- 极少字幕（整条内容可能只有 1-2 句）
- 像"内心备注"的短句（"今天没有崩溃，只是一直没空掉下来"）
- 小票体 / 价签体 / 洗标体（服装行业的"文本皮肤"）
- 留白收尾句（"我先允许自己软一点"）
- **避免**："你要爱自己 / 熬过去就好了 / 你已经很棒了 / 允许自己慢一点"等 AI 腔金句库

### B7 时间元素
- 一镜到底的真实时长（30-60 秒不剪）
- 深夜 / 清晨（情绪沉淀时段）
- 归家瞬间（下班到家的那一刻）
- 单一时刻的停驻（椅子旁坐 10 秒 / 窗前站 5 秒）

### B8 反差元素
- 外表正常 vs 内心悬着
- 动作熟练 vs 节奏慢了半拍
- 物件摆放整齐 vs 主人心里乱
- "今天没有崩溃" vs "只是一直没空掉下来"
- 被当作"还好"vs 实际"已经一整天憋着"

### B9 隐喻元素
- 外套 = 情绪防御层 / 安全感道具
- 工牌 = 社会角色的脱卸
- 椅背上的衣服 = 今天没被接住的情绪容器
- 挂衣架 = 把某种情绪暂时放回去
- 关灯 = 今天的句号
- **衣服不是商品，是"替主人活着"的那部分**

### C2 进阶玩法（少数高手在用）

> **本档首次应用 β 层「真实感三问自检」字段**——每张卡必须回答 3 个锚点（画面 / 情感 / 外部）。这是 emotion_expression 的**生命线防御**，不是装饰性字段。同时保留 product_review 首创的 β 标签方案 —— 「适用场景」= 引流 / 接客 / 两者通用。

#### 玩法卡 #C2-1 —— "脆弱但克制的自我披露 / 具体某一刻崩了一下"

| 字段 | 内容 |
|---|---|
| 档位 | ⭐ 进阶 |
| **适用场景** | **引流为主**（真实披露是第一层引流的强钩子 ; 接客场景下员工不适合对客人披露自己的脆弱）|
| 一句话定义 | 不是"我好惨"，而是"**我具体在哪一刻崩了一下，以及我怎么处理的**"。把脆弱从"情绪洪水"变成"可被理解的经验"——观众不需要承担你的情绪，只需要在其中找到自己的影子 |
| 北极星映射 | ✅ 观众不是"被你感动"，而是"**突然在你身上看到自己上周也这样过**"——这是比鸡汤高一个量级的共情 |
| 表达元素清单 | B4 真实生活场景（厨房 / 地铁口 / 楼梯间）/ B2 环境音 + 短句旁白 / B3 单人 / B6 少量字幕像内心备注 / B9 服装作为防御层 / B9 服装作为安全感道具 |
| **真实感三问自检** | ① **画面**：是不是**真的**那一刻的厨房 / 地铁口 / 楼梯间？（不是摆拍的 / 不是从素材库调的）② **情感**：是不是**你此刻**的情绪，还是"为了拍内容构思的情绪剧本"？③ **外部锚点**：有没有真实时间 / 真实物件 / 真实天气作为可验证的锚？ → **任一答"不"即不应作为 emotion_expression 发布** |
| 30 秒分镜参考 | **0-6 秒：** 回家进门 + 把包放在玄关地上 + 镜头不拍脸只拍手 + 钥匙声 \| **6-15 秒：** 走到厨房 + 站在水槽前没开水 + 停 4 秒 + 字幕"今天 3 点我在会议室快 哭 出 来" \| **15-22 秒：** 手开冰箱 + 看了一眼 + 又关上 + 字幕"然后我去洗手间冲了把脸回来开下一个会" \| **22-30 秒：** 坐在餐椅上 + 慢慢脱外套 + 字幕"现在我终于能承认 我今天不好"|
| 服装行业钩子 | ⭐⭐⭐⭐ 服装在这里是**情绪防御层** —— 外套的包裹感 / 帽衫的遮蔽感 / 衬衫的"社会盔甲" —— 服装变成了"今天没人陪你时替你活着的那一层" |
| 实拍可行性 | 🟢 1 人 1 手机可拍 / 预算 0 / 不依赖专业 / 4 小时内完成 / **时段重要：下班到家后的前 10 分钟是真实黄金窗口**|
| AI 辅助度 | ⚠️ 能做：叙事节奏建议、留白时点建议、"像内心备注"的字幕结构 ; ❌ 不能做：**替代"那一刻真的崩了一下"的真实性** —— 一旦是 AI 编的，就是 C1-3 的翻车 |
| 来源 | deep-research §emotion_expression 进阶 1 "脆弱但克制的自我披露"（学术研究"在线脆弱表达如何影响真实性与吸引力"引用）|
| 备注 | **最危险的翻车点**：把"具体哪一刻"替换成"抽象的一天"。具体时间 / 具体地点 / 具体事件是 β 层外部锚点的强硬要求 |

#### 玩法卡 #C2-2 —— "不讲情绪，讲物证 / 情绪被物件替你说出口"

### C3 突破性玩法（笛语平台主推）

> 本档是笛语对 emotion_expression 的核心交付物。**服装行业情绪表达的突破性不在"文案更美"，而在"让衣服替主人说完今天没说出口的话"**。每张突破性玩法卡都必须回答 β 层「真实感三问自检」。共 5 张。

#### 玩法卡 #C3-1 —— "一件衣服替我说完今天没说出口的话"

| 字段 | 内容 |
|---|---|
| 档位 | 🌟 突破性 |

| 实拍可行性 | 🟢 完全可拍 / 一部手机 + 家里一把椅子 / 预算 0 / 时段重要：下班到家后前 15 分钟 |
| AI 辅助度 | ⚠️ 能做：动作顺序建议、环境音清单、结尾字幕候选 ; ❌ 不能做：**替代"真实归家时那种说不清的悬浮感"** —— 这是 AI 腔最容易翻车的一层 |
| 来源 | 通用类.md §5 突破 1（原创设想）|
| 备注 | **本 CT 和笛语平台的旗舰玩法**——它同时验证了三件事：(1) 服装作为情绪载体的叙事纵深 ; (2) 共享层 #10 纯沉默叙事在通用类的适用性 ; (3) β 真实感三问自检的实际可操作性 |

#### 玩法卡 #C3-2 —— "不是我在讲情绪，是空间在替我呼吸"

|---|---|

#### 玩法卡 #C3-3 —— "衣柜情绪谱 / 不穿只挑，把情绪投射到衣服上"

| 实拍可行性 | 🟢 1 人 1 手机可拍 / 预算 ≤ 200 / 不依赖专业 / 4 小时内完成 |

## D. 服装行业垂直机会

情绪表达类内容，服装行业的**最大未开发机会**不是再拍"穿搭治愈系"，而是把服装当作**情绪的接口**：
- ❌ "这件针织很温柔" → ✅ "有些人会把外套留在你椅背上"
- ❌ "这件衬衫很有氛围" → ✅ "你今天不想讲话，但它还算体面地替你活着"

### 三个未被开发的角度

1. **"衣服作为情绪容器 / 防御层 / 陪伴物"**（C2-1 脆弱披露 / C2-2 物证 / C3-1 替我说完没说的话 / C3-3 衣柜情绪谱）——服装是**其他行业没有的"第二皮肤"** ——既是社会身份又是情感保护，这是内容叙事的独有空间
2. **"门店 / 试衣间 / 后台作为情绪空间"**（C3-2 空间替我呼吸）——服装门店的打烊后 / 后台 / 试衣间是**极稀有的"半公共半私密"场景**，可以承载"动作变慢"的情绪纪录
3. **"服装行业的文本皮肤（小票 / 价签 / 洗标）"**（C3-4 文本皮肤）——这是**其他品类完全没有的文字载体** ——可以做成一个长期的"情绪翻译系列"

### 服装行业 emotion_expression 标杆案例

| 账号 / 品牌 | 平台 | 数据 | 走的路线 |
|---|---|---|---|

### 行业趋势支撑

- **叙事沉浸研究**：故事邀请受众进入叙事世界 / 人会被这种经历暂时改变 / 当人沉浸于故事时态度与意图更可能与故事一致
- **Kuleshov 效应（电影认知学）**：同一张中性脸在不同情境镜头夹击下会被观众读出不同情绪
- **沉默 vlog 全球大趋势**：85% 的社交视频是**静音观看**的 ; 沉默 vlogger 的观看量增长 300% ; 平均互动率 12% vs 语音型 4-6%
- **2024 中国青年消费洞察**：**46.28% 的年轻消费动机是"取悦自我"** —— 情绪资产（Emotional assets）成为品牌留存与溢价的设计方向
- **加速暴力批判**：深度情感体验被扁平化为转瞬即逝的神经冲动 —— 学术界对情绪内容 AI 化的最严厉批评

## E. 红线（一定不行的玩法）—— 含 α 层「9 种 AI 腔禁用清单」

### E-α 「9 种 AI 腔禁用清单」——负面样板库（强制禁用 / 不是提醒）

> **使用方式**：AI 在生成 emotion_expression 时，先做负面匹配——如果生成结果命中下面任一模式，必须**重写**而非微调。对人工创作者，这是"**一旦你写的内容长这样，立刻删掉重来**"的自查清单。

| # | AI 腔模式 | 典型样板句 / 样板结构 | 为什么致命 |
|---|---|---|---|
| **E-α1** | 泛用抚慰型 | "允许自己慢一点 / 你已经很棒了 / 做自己就够了 / 你要爱自己" | Barnum effect —— 对所有人都适用 = 对每个具体的人都不适用 |
| **E-α2** | 凌晨三点诗意型 | "凌晨三点的城市 / 只剩下一个人的孤独 / 月光打在窗边 / 像极了爱情" | 高密度 AI 训练数据 —— 每个词都可替换 / 每句都可互换位置 |
| **E-α3** | 成年人崩溃型 | "成年人崩溃就在一瞬间 / 有时候不是累 是没人懂" | 结论先行 / 没有具体情境 / 被 C1-1 玩到烂 |
| **E-α4** | 反问灌输型 | "为什么你总是 XX? / 难道你不配 XX 吗? / 多久没 XX 了?" | 强迫观众情绪 / 等于道德绑架 / AI 最爱用的结构 |
| **E-α5** | 排比鸡汤型 | "你可以不完美 / 你可以不优秀 / 你可以不努力 / 但你不能 XX" | 三排比 + 最后反转是 AI 语言模型的偏好路径 / 一秒识别 |
| **E-α6** | 隐喻过载型 | "人生就像一杯茶 / 苦尽甘来是真味 / 而我还在等水开" | 每句都想当金句 / 堆砌隐喻 / 完全没有具体画面 |
| **E-α7** | 空洞励志型 | "坚持下去 / 风雨过后就是彩虹 / 你比你想象的更强大" | 无对象 / 无时间 / 无地点 / 纯鼓励 / AI 默认路径 |
| **E-α8** | AI 配音腔 | "合成女声 + 刻意悲伤 + 尾音拖长 + 每个词都读得过于清晰" | 不是文字问题 / 是**声音本身**就是 AI 腔 —— 一秒识破 |
| **E-α9** | 滤镜 + BGM 包装型 | 黑白滤镜 + 悲怆钢琴 + 慢动作 + 眼眶湿润特写 | 所有视听要素都在"暗示观众该哭了"—— 观众的 AI 免疫反应立刻启动 |

**用法**：本清单不是"建议避免"，而是**AI 生成 emotion_expression 时必须执行的硬约束**——命中任一项则拒绝生成，要求重构。对人工创作者，自查清单挂在 F 段 AI 辅助边界里作为"AI 不能做"的具体例证。

## F. AI 辅助边界

| 环节 | AI 能做 | AI 不能做 |
|---|---|---|
| 选题 | 物件清单（按情绪状态生成）/ 生活场景建议 / 情绪 - 动作映射 / 情绪 - 衣物映射 | 替代"你今天真实的情绪是什么"|
| 脚本 | 叙事结构建议（起承转合）/ 字幕候选（需人工筛选）/ 小票 / 价签 / 洗标文案候选 | 生成**任何**命中 §E-α「9 种 AI 腔禁用清单」的句式（硬禁用）|
| 拍摄 | 镜头清单 / 固定机位建议 / 环境音收录建议 / 色温渐变方案 | 替代真实生活场景 / 真实物件 / 真实声音 |
| 剪辑 | 节奏慢度建议 / 一镜到底保留建议 / 字幕减法（不是加法）| 决定"删掉哪些解释 / 保留哪些欲言又止"|
| 发布 | 时间标签 / 话题候选 / 非 AI 腔标题（通过 §E-α 负面匹配过滤）| 替代"这个情绪是不是你此刻的" |

> **F 核心判断：** **谨慎辅助**——能做结构 / 清单 / 负面过滤 ; 但**"情绪本身必须来自创作者自己的真实生活与门店现场，AI 不能代演"**。**本 CT 的 AI 辅助度是整个 Q2 里最低的**——不是因为 AI 能力不够，而是因为情绪内容一旦被识别为"AI 生成的"就彻底失效。β 层真实感三问自检必须在 AI 生成流程里作为**强制校验前置**——任一项答"不"，生成拒绝。
