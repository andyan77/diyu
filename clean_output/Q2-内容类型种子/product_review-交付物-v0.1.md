## 本文件是什么（核心定义，防止脱离索引被误读）

- **交付物 ≠ 字段 schema**。交付物是一份"**玩法库 + 元素库 + 红线**"，是 AI 生产这种内容时可调用的**表达资源包**。
- **模板 ≠ 文字填空模板**。每张玩法卡里的「30 秒分镜参考」就是模板——**拍摄者照着能拍出来的执行卡片**，含画面 + 声音 + 动作 + 节奏。
- **AI 生产时不是"填字段"**，而是"**从玩法库里选一张卡片并展开**"。

> 定义来自用户 2026-04-13 裁决。详见 [_index.md](_index.md) §0。

## 本文件与 batch 1/2/3 的关键差异：双角色「适用场景」标签机制（首创）

### 1. 为什么 product_review 需要「适用场景」标签

Q1 v1.1 明确 `product_review` 同时出现在三层框架的**第一层引流**和**第二层接客**中（[phase-3-Q1-内容生产首批做什么.md](../phase-3-Q1-内容生产首批做什么.md) §第一层 + §第二层）。两层的业务逻辑完全不同：

| 维度 | 引流向（第一层） | 接客向（第二层） |
|---|---|---|
| 观众 | 陌生人（公域刷到）| 进店犹豫的顾客 |
| 核心任务 | 建立第一印象 + 抓眼球 + 种草 | 消除顾虑 + 推荐对味 + 转化 |
| 生产主体 | 加盟商 / 主理人自拍 | 加盟商对店内顾客演示 |
| 节奏 | 紧凑（30-60 秒）| 可稍长（60-120 秒甚至即兴）|
| 最危险的陷阱 | 形容词堆叠 / AI 味 | 只讲优点 / 像推销 |

### 2. β 标签方案（用户 2026-04-14 裁决）

**每张进阶 / 突破性玩法卡都含「适用场景」标签**，取值为：
- **引流**：主要服务第一层，陌生观众转化为关注者
- **接客**：主要服务第二层，店内顾客当场决策
- **两者通用**：玩法本体可在两种场景下同时使用

**核心原则**：玩法本体（例如"触感法庭 / 只拍犹豫 / 反滤镜宣言"）是**跨场景共享的**，差异是执行情境。不为同一玩法写两张不同的卡——信息会爆炸。

### 3. 与 `outfit_of_the_day` 的区别

batch 1 的 `outfit_of_the_day`（穿搭日记）也涉及试穿，但叙事重心是"**今天这一套**"——展示自己的日常穿搭选择。本 ContentType 的叙事重心是"**这件是否值得买**"——对具体商品的判断和推荐。同样拍试穿镜头，前者是创作者个人的穿搭日记，后者是对商品的可信度证据。

## A. 北极星

**观众看完后，不是"知道这件衣服有什么参数"，而是"**我已经在脑子里提前穿过一次、犹豫过一次、相信过一次**"——风险已经看清、信任已经建立、购买决策的心理阻力已经被消解。**

> **三份原话备查：**
> - 通用类.md：在脑子里提前穿过一次、犹豫过一次、相信过一次
> - 通用类compass.md：reposition garments as identity carriers, emotional artifacts, and narrative devices — 不是把衣服当商品，而是当情绪载体 / 身份容器
> - 通用类deep-research-report.md：我已经把风险看清了 / 值不值 / 怎么买最不踩坑 / 我也愿意相信这个人

## B. 元素库（原子层）

### B1 视觉元素
- 全身试穿镜头（但**避免转圈** —— 转圈是 C1 陷阱的标志）
- 动作镜头（走路 / 坐下 / 抬手 / 弯腰 / 系扣子 / 拉拉链）
- 手部细节特写（捏布料 / 拉弹性 / 摸车线 / 翻洗标）
- 微距面料镜头（手机近拍可达 ≤ 5cm / 可选 ¥30 手机微距夹镜头）
- 逆光 / 侧光测试（测透光 / 测面料厚度 / 测颜色在光下的表现）
- 镜前 + 侧拍双角度（同一个动作从两个视角拍）
- 生活场景碎片（地铁 / 楼下便利店 / 办公椅 / 餐桌 / 沙发）
- 同类商品并排对比（同色 / 同款 / 同价位）
- 桌面俯拍（物证剧场 / 触感法庭构图）
- **避免**：正面转圈全身镜 / 快切换装蒙太奇 / 滤镜空镜氛围

### B2 听觉元素
- 布料摩擦声（走路时 / 坐下时 / 抬手时）
- 拉链声 / 扣子碰撞声 / 魔术贴撕开声
- 弹性测试声（捏拉布料的轻响）
- 走路脚步声（真实地板 / 地铁站 / 商场）
- 环境音（地铁广播 / 办公室键盘 / 餐厅背景声 / 家中空调声）
- 极少 BGM 或完全无 BGM
- 低声口播（不是"姐妹们冲"式兴奋语气，而是冷静描述）
- **避开**：热门 BGM 快切 / 夸张惊叹语气 / "好绝 / 谁懂"口头禅

### B3 人物元素
- 创作者本人试穿（默认）
- 家人 / 同事 / 伴侣 / 店员作为**第二评价者**（关系型测评）
- 店内真实顾客（接客场景，需同意）
- 只露手 / 只露身体部位（无面部测评）
- 同性别 + 同身材友人作为对照组（多人试穿同款）

### B4 场景元素
- 镜前 / 试衣间（基础场景）
- 窗边自然光（反滤镜宣言的关键）
- 真实生活场景（通勤路 / 办公室 / 餐厅 / 家中沙发 / 地铁）
- 门店陈列区（接客场景下的第二层定位）
- 桌面 / 工作台（物证剧场构图）
- 极端对照场景（早市 → 办公室 → 建筑工地 → 酒吧）
- 同一件衣服在 3-5 个场景中出现

### B5 情绪元素
- 克制感（不夸张 / 不安利 / 不一上来给结论）
- 犹豫感（真实的"这里我有点不确定"）
- 冷静判断感（像法官而非销售）
- 反向透明（"这里有点翻车"的坦承）
- 顾虑先行（"我本来怕它透 / 怕它勒 / 怕它扎"）
- 延迟结论（30-40 秒后才给判断，前半程只做测试）

### B6 文字元素
- 极少字幕（只标注关键测试点 / 单位 / 时间）
- 数据标签（"克重 180g" / "坐 20 分钟" / "回弹 0.8 秒"）
- 证物标签风格（"透光 ✕" / "起静电 ✓" / "起球风险 中"）
- 顾虑清单（"怕透 / 怕勒 / 怕皱 / 怕扎 / 怕短"）
- 留白式结论（"如果你和我怕的是同几件事，这件可以进试衣间"）
- **避免**："绝绝子 / 必买 / 冲就完了 / 闭眼入"

### B7 时间元素
- **一天体感**（早 / 午 / 下午 / 晚四个时刻同一件衣服）
- **三次切片**（同一件衣服在同一天被误会三次）
- 坐 20 分钟后的褶皱恢复测试
- 抬手 / 弯腰 / 捡东西的瞬间定格
- 慢动作特写（布料下落 / 裙摆摆动 / 缝线拉伸）

### B8 反差元素
- "别人看我" vs "我穿着它的身体感受"（双层字幕）
- 开箱瞬间感受 vs 穿一天后的感受
- 镜前审美 vs 坐下起身的真实动作
- 形容词评价 vs 可验证细节（审美 vs 证据）
- 预期翻车点 vs 真实表现（"本来怕 X / 其实 Y"）

### B9 隐喻元素
- 服装 = 身体界面（同时是社会符号和体感容器）
- 试穿 = 购买前的"心理预演"
- 面料 = 材料学证物（微距下可验证）
- 洗标 = 真实成分承诺（不会说谎的字段）
- 时间褶皱 = 一天生活的沉淀证据

## C. 玩法库（核心交付物）

### C1 主流玩法（默认禁用 / 低优先）

> 本档玩法是**整个短视频生态里最饱和的类型**——服装试穿测评已被 MCN 机构化生产。三份原料均点名这些模板是"AI 最容易复制 / 观众已经免疫 / 信任资产最弱"的陷阱。笛语平台对 product_review 的 C1 档**默认禁用**。

#### 玩法卡 #C1-1 —— "怼脸报身高体重 + 转圈念优缺点 / 三连快评"

| 字段 | 内容 |
|---|---|
| 档位 | 🚫 主流 / 默认禁用 |
| 一句话定义 | 镜头一开，先报身高体重尺码，然后在镜头前左右转前后转，最后总结"显瘦 / 遮胯 / 面料舒服 / 微胖友好"。像把商品说明书念成了口播。也常见变体："开箱 → 上身 → 一句结论 / 姐妹冲"|

### C2 进阶玩法（少数高手在用）

> **本档是第一次应用「适用场景」β 标签方案**——每张卡标注"引流 / 接客 / 两者通用"。玩法本体跨场景共享，差异是执行情境。

#### 玩法卡 #C2-1 —— "体感测评 / 不先下结论"

| 字段 | 内容 |
|---|---|
| 档位 | ⭐ 进阶 |
| **适用场景** | **两者通用**（引流：区别于形容词堆叠的高级感 ; 接客：店内直接让顾客做动作测试）|
| 一句话定义 | 不上来就夸，把"穿上后的身体反馈"放在最前面——走路时裤腿扫脚背吗 / 坐下时腰头顶胃吗 / 抬手时肩线拽吗。把"评价"从审美判断拉回到身体经验 |
| 北极星映射 | ✅ 不把观众当"看图做梦的人"，而是把观众当"即将花钱真实穿它的人"——信息密度更高、更难被空洞形容词替代 |
| 表达元素清单 | B1 手部触感特写 / B1 坐下抬手走动测试 / B6 少形容词 / B1 动作反馈 / B2 局部收音 / B4 自然光 / B1 镜前 + 侧拍结合 |
| 30 秒分镜参考 | **0-6 秒：** 镜前穿上新衣 + 低声"还没开始走，先试试抬手" + 抬手镜头 + 肩线特写 \| **6-14 秒：** 坐到沙发 + 腰头特写 + "这里有点顶" + 停 2 秒再站起 \| **14-22 秒：** 走几步 + 裤腿与脚背特写 + 摩擦声 \| **22-30 秒：** 镜前停 + "坐下不顶 / 走路不扫 / 肩线正好" + 字幕"这 3 个动作决定我会不会买它"|
| 服装行业钩子 | ⭐⭐⭐⭐ Shopify 电商文档明确指出："不要停在'高质量'这种泛词，要写清场景、细节、穿着感受"——**服装的体感测评是其他品类无法替代的内容护城河**|
| 实拍可行性 | 🟢 1 人 1 手机完全可拍 / 预算 0 / 不依赖专业 / 4 小时内完成 |
| AI 辅助度 | ✅ 能做：动作测试清单、镜头顺序建议、字幕减法 ; ❌ 不能做：替代真实的身体反馈 |
| 来源 | 通用类.md §1 进阶 1 + deep-research §product_review 进阶 1 "原相机反滤镜 + 细节显微镜" 合流 |
| 备注 | "反滤镜宣言"是进阶测评的信任地基——原相机 / 自然光 / 不拉腿 / 不美颜四项自述一旦缺失，下面的所有体感测评都失去可信度 |

#### 玩法卡 #C2-2 —— "关系型测评 / 让第二个人来拆穿你"

| 字段 | 内容 |
|---|---|
| 档位 | ⭐ 进阶 |
| **适用场景** | **引流为主**（关系感是第一层引流的强钩子 ; 接客场景下店员和顾客的关系已存在，无需额外制造）|
| 一句话定义 | 创作者自己试穿，同时让妈妈 / 同事 / 伴侣 / 店员在旁边给出"不中听但有用"的反应——"这个肩太精神了，有点像去开会" / "这个颜色本人比镜头里更显脸白"。单人测评容易变成自说自话，第二个真实关系人的介入把"审美判断"变成"社会判断"|

### C3 突破性玩法（笛语平台主推）

> 本档是笛语对 product_review 的核心交付物。**服装测评的突破性不在"拍得更好看"，而在把评价从语言搬到感官、从结论搬到过程、从审美搬到体感**。AI 生产 product_review 时应该**优先**从 C3 取卡。共 5 张。

#### 玩法卡 #C3-1 —— "一天只穿这一件 / 不急着夸"

| 字段 | 内容 |
|---|---|
| 档位 | 🌟 突破性 |
| **适用场景** | **引流为主**（一天叙事是第一层引流的强钩子 ; 接客场景单次只有 5-30 分钟，无法覆盖一天时间跨度）|
| 一句话定义 | 早上出门前架手机在镜前 + 第一句"今天我不换了，就穿它一天"。接下来不是连拍，而是三到四个生活切片：地铁里站着时扯了一下裤腰 / 中午坐下吃饭时摸了一下肚子位置 / 下午进试衣间补妆时看了一眼肩线 / 晚上回家把包放下，裙摆有一点生活褶。**最后一句不是"值得买"，而是"到晚上我还没想把它脱掉"**|
| 北极星映射 | ✅ 让衣服接受**时间考验**，而不是接受**口才考验**——观众接住的不是"这件衣服很棒"，而是"这件衣服能陪你过一天，而且不烦你"。这比显瘦更真实 |
| 表达元素清单 | B7 时间切片 / B7 生活褶皱 / B1 镜前自检 / B6 少量字幕 / B2 环境音 / B5 延迟结论 / B7 夜晚收尾 |
| 30 秒分镜参考 | **0-5 秒【早 7:30】：** 镜前穿上 + 字幕"今天我不换了 就穿它一天" + 无口播 \| **5-12 秒【中午 13:00】：** 办公室座位上 + 坐下时手下意识摸一下肚子位置 + 没有抱怨也没有夸赞 \| **12-20 秒【下午 17:00】：** 洗手间镜前 + 自己整理一下肩线 + 镜头给肩膀特写 \| **20-27 秒【晚 22:00】：** 回家放下包 + 镜头拍到裙摆有一点褶 + 字幕"生活褶 · 第一件给我的"\| **27-30 秒：** 直视镜头 + "到晚上 我还没想把它脱掉"|
| 服装行业钩子 | ⭐⭐⭐⭐ 服装在这里不是被展示的对象，而是"**陪伴一天的共同主演**"——尤其适合通勤裤 / 衬衫 / 针织 / 内搭等"每天都要穿但不张扬"的品类 |
| 实拍可行性 | 🟢 1 人 1 手机可拍 / 只拍一天中的 3-4 个时刻 / 无额外成本 / 可能需要跨时段 4-8 小时但纯拍摄时间 < 30 分钟 |
| AI 辅助度 | ✅ 能做：生活切片清单、时间段建议、字幕留白建议 ; ❌ 不能做：替代真实的一天生活褶 |
| 来源 | 通用类.md §1 突破 1（原创设想）|
| 备注 | 这个玩法的信任来自**时间**——观众知道你真的穿了一天，而不是在同一个试衣间里摆拍 4 次 |

#### 玩法卡 #C3-2 —— "只拍犹豫 / 不拍优点 / 顾虑解除测评"

| 字段 | 内容 |
|---|---|
| 档位 | 🌟 突破性 |
| **适用场景** | **两者通用**（引流：反套路种草 ; **接客尤其强**——进店顾客的犹豫点就是现场可拍素材）|
| 一句话定义 | 整条视频只讲犹豫点——这件白衬衫我本来怕透 / 这条裤子我本来怕显胯 / 这件针织我本来怕扎。每个"怕"都给一个小测试：窗边逆光看透不透 / 侧身站着看胯 / 手臂贴脸颊看扎不扎。**前 20 秒都没有说"好"**，只是一直验证"原来没我想的那么糟"。最后留一句"如果你和我怕的是同几件事，这件可以进试衣间"|
| 北极星映射 | ✅ 把成交路径从"被种草"改成"被安慰"——观众接住的是购买时最真实的心理活动：不是想象自己有多美，而是在判断"会不会踩雷"|
| 表达元素清单 | B8 反向提问 / B1 局部测试 / B1 逆光 / B1 侧身 / B5 停顿 / B6 弱结论 / B6 顾虑清单 |
| 30 秒分镜参考 | **0-4 秒：** 镜头正对 + "这件白衬衫我本来怕透"+ 字幕"怕 1 ：透"\| **4-10 秒：** 走到窗边逆光 + 抬手让光穿过 + 字幕"逆光 ✕ 没透" \| **10-16 秒：** "这条裤子我本来怕显胯" + 侧身镜前站 2 秒 + 字幕"侧身 ✕ 不显"\| **16-22 秒：** "这件针织我本来怕扎" + 手臂贴脸颊 + 字幕"扎 ✕ 不扎" \| **22-30 秒：** 字幕"我本来怕的 3 件事 都没发生 / 如果你和我怕的一样 它可以进试衣间"|
| 服装行业钩子 | ⭐⭐⭐⭐⭐ 服装太适合做"顾虑清单解除"——消费者真正卡住的就是**透 / 勒 / 皱 / 闷 / 短 / 显壮 / 难搭**，这是服装独有的"心理阻力图谱"|
| 实拍可行性 | 🟢 完全可拍 / 手机 + 一个窗边 / 预算 0 / 4 小时内完成 |
| AI 辅助度 | ✅ 能做：顾虑清单（按服装类型生成）、测试动作建议 ; ❌ 不能做：替代真实的"我本来怕 X"的第一人称视角 |
| 来源 | 通用类.md §1 突破 2（原创设想）|
| 备注 | **接客场景的专属用法**：店员站在店内顾客旁边，拿出同款现场做 3 个顾虑测试 —— 顾客"我怕这个肩太硬"→ 店员让她侧身在镜前看 2 秒 → 真实反馈。这是笛语在接客场景下最强的转化工具 |



#### 玩法卡 #C3-3 —— "触感法庭 / 同类三件对照声音证据"

| 字段 | 内容 |
|---|---|
| 档位 | 🌟 突破性 |
| **适用场景** | **接客为主**（店内场景感最强 ; 引流场景也成立但同类三件的布局更适合门店）|

| 北极星映射 | ✅ 把"质感"变成**听得见、看得见的证据**——观众不是被你说服，而是自己在大脑里完成"触感想象"，从而更确信"这件衣服和我的身体会怎么相处"|
| 表达元素清单 | B1 俯拍 / B1 微距 / B1 慢动作 / B1 同机位对比 / B2 衣料声为主 / B2 几乎无 BGM / B3 只露手 / B4 桌面 / B6 证物标签风格字幕 |

| 30 秒分镜参考 | **0-10 秒：** 俯拍桌面 + 三件同色白 T 并排 + 手指一件件捏住布料角 + 低头揉搓 + 字幕"回弹 / 垂坠 / 发硬" \| **10-18 秒：** 三件依次做领口拉伸测试 + 画面出现小计时器 + 每件停 2 秒 \| **18-25 秒：** 手把第一件贴近麦克风 + 轻轻划过 + 布料声特写 + 切换到第二件 + 再换第三件（声音差异明显）\| **25-30 秒：** 镜头停在第 2 件 + 字幕"如果你讨厌起球 今天只选 2 号"|

| 备注 | **接客场景最强用法**：店内两个犹豫的客人选两件相似品，店员现场做 30 秒触感法庭——声音和触感胜过一切口播，这是店内最强转化工具 |

#### 玩法卡 #C3-4 —— "无声叙事试穿 / 衣橱考古"

| 字段 | 内容 |
|---|---|
| 档位 | 🌟 突破性 |
| **适用场景** | **引流为主**（艺术化 / 电影感 / 反套路 / 最容易形成平台爆点）|
| 一句话定义 | 两种变体合流——**变体 A 无声叙事试穿**：无口播、无热门 BGM，只有环境音（风 / 脚步 / 布料摩擦 / 门关上）。镜头跟随一个人穿着新衣做日常活动，每个场景 5-7 秒。服装**从不被"介绍"**，只是存在于运动的身体上。**变体 B 衣橱考古**：开场是衣橱深处一件被遗忘的旧衣，镜头像考察文物一样检查缝线 / 标签 / 褪色 / 质感 ; 然后把新买的那件放在旁边——"这件新的，配得上我最珍爱的旧的吗？"|
| 北极星映射 | ✅ 把服装从**商品**转变为**自传**——新衣不是"被推荐"，而是"被接纳" ; 观众接住的是一种"生活里的选择"，而不是"货架上的选购"|
| 表达元素清单 | B1 iPhone 电影模式（选择性聚焦布料纹理）/ B1 黄金时段光线 / B2 环境音 / B6 极少字幕 / B4 真实生活场景 / B1 微距面料特写 / B5 克制感 |
| 调用共享引擎 | #10 纯沉默叙事（Burberry / Bottega Veneta 同路径 / 店内商品叙事的重要共享引擎）|
| 30 秒分镜参考 | **0-6 秒：** 衣橱深处拿出一件旧毛衣 + 特写褪色的袖口 + 无 BGM 只有衣架声 \| **6-12 秒：** 手指摸一摸旧毛衣的磨损处 + 放到一边 + 拿出新买的那件（同色系）\| **12-18 秒：** 镜头跟随你穿着新衣走出门 + 黄金时段光线 + 脚步声 \| **18-25 秒：** 公园长椅坐下 + 手摸新毛衣袖口 + 镜头切换到旧毛衣仍在家里的画面 \| **25-30 秒：** 回家 + 把新毛衣挂到旧毛衣旁边 + 字幕"今天它们第一次打招呼"|

| 字段 | 内容 |

## D. 服装行业垂直机会

服装做 `product_review` 的**最大机会**不是"再多做几条试穿"，而是把服装从"**视觉商品**"重新拉回"**身体体验商品**"。

### 三个未被开发的角度

1. **"体感测评作为反 AI 护城河"**（C2-1 体感测评 / C3-1 一天只穿 / C3-2 只拍犹豫）——你这个平台如果只会生成"显瘦 / 遮肉 / 百搭"，永远会像批量 AI ; 但如果它能生成"逆光测试 / 抬手测试 / 坐下测试 / 一天体感记录 / 顾虑解除脚本"，它就开始有行业理解了
2. **"面料 / 工艺 / 辅料的微观叙事"**（C2-3 面料侦探 / C3-3 触感法庭）——服装是最多"隐性雷点"的品类（起球 / 缩水 / 静电 / 褪色 / 透光），把这些拍成可验证证据链是其他品类无法复制的内容护城河
3. **"时间 / 情境作为叙事维度"**（C3-1 一天只穿 / C3-4 衣橱考古 / C3-5 情绪盲测 / Kuleshov）——服装是最容易与**时间**和**情境**建立关系的商品，通过时间流逝或情境切换来展示价值，是其他零售品类没有的叙事纵深

### 服装行业 product_review 标杆案例

### 行业趋势支撑

- **Shopify 电商文档**：写服装描述时要写清场景 / 细节 / 穿着感受 / fit 与使用情境，不要停在"高质量"这种泛词
- **Interface Fashion 报告**：MCN 管理的测评博主造成严重同质化，观众对"多位博主同评一物"会自动归类为"协调推广"
- **Vogue 对 haul / try-on 的观察**："keep or return"已成常见提示语 —— 互动钩子模板化到失去真实性
- **TIME 报道**：TikTok 上 deinfluencing（反种草）作为对过度带货的反弹，核心是质疑"你真的用过吗"
- **营销学 ASMR 研究**：unboxing 里的"无语段落 + 物体声"段落能触发类 ASMR 吸引力

## E. 红线（一定不行的玩法）

| # | 红线 | 来源 |
|---|---|---|
| E1 | **用一串审美形容词代替真实测试** —— "高级 / 绝美 / 显气质 / 谁懂" —— 所有词都可替换到另一件衣服 | 通用类.md §1 红线 1 |

| E3 | **明明是测评，却全片只拍滤镜氛围，不拍细节缺点** —— 滤镜遮羞布 | 通用类.md §1 红线 3 |
| E4 | **一上来"闭眼入、冲就完了"** —— 没有任何顾虑解除 —— 观众的第一反应是"这是广告" | 通用类.md §1 红线 4 |
| E5 | **为了"真诚"强行演夸张惊喜** —— 反而更像脚本 | 通用类.md §1 红线 5 |
| E6 | **"全优点无缺点"的测评** —— 在合作生态里差评稀缺会直接引发真实性质疑 | deep-research §product_review 红线 2（界面新闻引用）|

| E9 | **为反而反的"假拔草"** —— CBNData 记录了"只做真实测评"的拔草博主翻车、抄袭、前后不一 —— 反种草的真实性门槛比种草还高 | deep-research §product_review 红线补充 |

## F. AI 辅助边界

| 环节 | AI 能做 | AI 不能做 |
|---|---|---|
| 选题 | 按服装类型生成顾虑清单（通勤裤怕勒 / 白衬衫怕透 / 针织怕扎）/ 测试动作清单 / 触感对比维度 | 替代"你今天真的穿了这件衣服的身体感受"|
| 脚本 | 顾虑解除三段式 / 体感测评节奏建议 / 反种草对比维度 / 字幕证物标签格式化 | 替代"这里真的勒 / 真的皱 / 真的翻车"的真实反馈 |
| 拍摄 | 镜头清单 / 机位建议 / 微距拍摄方案 / 逆光时间点建议 | 替代真实的面料手感和动作反馈 |
| 剪辑 | 三段式节奏 / 字幕减法 / BGM 候选（多数建议无 BGM 或极弱环境音）| 决定"删掉哪些掩饰 / 保留哪些犹豫"|
| 发布 | 标题模板（"本来怕 X / 其实 Y"）/ 话题标签 / hashtag 组合 | 替代"这个人是不是真的会穿这件"的人格感 |

> **F 核心判断：** **可深度辅助**——能做顾虑清单、测试动作设计、证物标签格式、对比维度 ; 但**"体感 / 犹豫 / 翻车的真实性"必须来自真人真实穿着，AI 不能代演**。**最容易翻车的是 C1 风格回流**——AI 天然偏向"形容词堆叠"和"正面结论"，必须在生成时主动抑制这两种倾向。
