## 本文件是什么（核心定义，防止脱离索引被误读）

- **交付物 ≠ 字段 schema**。交付物是一份"**玩法库 + 元素库 + 红线**"，是 AI 生产这种内容时可调用的**表达资源包**。
- **模板 ≠ 文字填空模板**。每张玩法卡里的「30 秒分镜参考」就是模板——**拍摄者照着能拍出来的执行卡片**，含画面 + 声音 + 动作 + 节奏，不是字段拼装。
- **AI 生产时不是"填字段"**，而是"**从玩法库里选一张卡片并展开**"。

> 定义来自用户 2026-04-13 裁决。详见 [_index.md](_index.md) §0。

## A. 北极星

**观众看完不只是"我听懂了"，而是"我终于被一个真实懂的人讲明白了，并且把你归类为可信、讲得清楚、愿意讲真话的人"。**

> **三份原话备查：**
> - GPT5.4：不只是"我听懂了"，而是"我终于被一个真实懂的人讲明白了，我愿意信他"
> - 深度研究：产生"啊原来如此"的顿悟，把你归类为"可信、讲得清楚、愿意讲真话的人"
> - compass：算法在奖励"有灵魂的内容"而非"有套路的内容"；知识分享最核心的区别是有没有"为什么是你来讲"

## B. 元素库（原子层）

### B1 视觉元素
- 桌面固定机位 + 手部特写（实验演示）
- 近景 / 中景切换（肩线 / 版型 / 布料）
- 前后对比镜头（错误 vs 正确）
- 个人经历场景插图（家族老照片 / 童年 / 转折点）
- 手绘 / 白板 / 小道具（喷壶 / 纸巾 / 灯 / 尺子）
- 衣物近拍（拉链、纽扣、走光点）
- 屏幕截图（合拍 / Stitch 原视频）

### B2 听觉元素
- 同期声（摩擦 / 水滴 / 光线 / 纸笔）
- 布料摩擦 / 拉链 / 撕扯 / 拉伸声
- 走路时裤脚摩擦声
- 领夹麦近场收音
- 夸张但可读的手势声响（ASMR 式）
- 完全无人声（纯展示）

### B3 人物元素
- 创作者本人真实经验
- 单人出镜讲解
- 只露手不露脸（纯桌面实验）
- 合拍 / Stitch 回应别人的问题

### B4 场景元素
- 桌面实验室（家里的桌子即可）
- 试衣间 / 自家衣柜
- 日常生活场景复现（错搭 / 洗护事故）
- 光源测试区（窗边 / 灯下 / 阳光下）

### B5 情绪元素
- 真诚而非表演性 hook
- 故事化（"我以前也搞错过"）
- 判断力 > 结论（授人以渔）
- 服务感 / 解决问题

### B6 文字元素
- 关键词字幕（只 3 个词）
- 误区/正确做法对照标签
- 极简人群/场景标签（"适合夏天 / 适合通勤 / 适合怕热的人"）
- 无完整口播（动作讲清一切）

### B7 时间元素
- 因果倒叙（先看后果，再拆原因）
- 过程完整展示（失败 / 重来 / 修正）
- 实验一次成型 vs 多次对比

### B8 反差元素
- 错误 vs 正确
- 低廉表现 vs 真实质感
- 静态展示 vs 动态使用
- 口播结论 vs 身体证据

### B9 隐喻元素
- 知识不是"老师在讲"而是"事故在发言"
- 知识长在手上（身体掌握 ≠ 嘴上念）
- 面料 = 声音（听觉代替视觉判断）
- 衣服 = 生活困惑的解释器

### C3 突破性玩法（笛语平台主推）

| AI 辅助度 | ✅ 能做：常见错误清单 / 因果拆解 ; ❌ 不能做：替代真实"我也搞错过"的坦诚 |

## D. 服装行业垂直机会

服装行业做知识分享，最有价值的不是"教你穿搭"，而是**把隐性经验显性化**：什么是好肩线、为什么门店灯光把衣服拍丑、为什么某种面料在通勤里比在静态展示里更值钱、为什么某件衣服上身后会改写一个人的行动方式。

### 三个未被开发的角度

1. **"声学面料知识"**——C3-5 材质听诊器是独占机会，把面料的"触感 / 声音"翻译成可传播的听觉记忆点
2. **"错误教学"**——C3-1 从一个错误开始讲，服装行业错误案例（错搭 / 错洗护 / 错版型）库存极大
3. **"动作证明型知识"**——C3-2 一句知识只靠动作讲清，服装行业的"肩线 / 垂感 / 走光 / 舒适度"都可以动作证明

### 服装行业知识方向（v0.2 M4 无痕处理：删去具体标杆案例品牌名 / 仅保留方向描述）
- **硬核面料 / 版型 / 缝线做工科普方向** —— 从工艺细节入手做"看起来枯燥但越看越上头"的硬知识
- **面料对比测评方向** —— 同类型面料的横向对比 / 显微视角 / 数据化呈现
- **"徒手做实验"方向** —— 用日常工具做面料/版型的物理验证（撕扯 / 浸泡 / 摩擦 / 对比缝合）/ 把"硬科学"包装成"硬手艺"

### 行业趋势支撑
- TikTok 2025 说平台已经成为 trusted insights 的入口
- YouTube 2026 把"帮助学习新技能""研究品牌/产品"写成核心使用行为
- 服装品牌若能把知识内容做成"被主动寻找的真实解释"，信任价值远大于一次种草

## 与试点版（personal_vlog）的差异点

1. **北极星完全不同：** personal_vlog 是"陪伴 + 好感"，knowledge_sharing 是"顿悟 + 信任"——一个感性，一个理性
2. **突破性玩法更少（6 张 vs 9 张）：** 因为知识分享的"创意空间"比 Vlog 小——知识本身是硬约束，不能为了突破牺牲准确性
3. **AI 辅助度更高：** 知识分享的逻辑骨架、误区清单、对比维度都可以 AI 辅助，比 personal_vlog（"真情绪"）依赖 AI 的程度更高
4. **服装行业钩子更直接：** 可以直接引出"面料 / 工艺 / 版型 / 洗护"卖点而不尬
