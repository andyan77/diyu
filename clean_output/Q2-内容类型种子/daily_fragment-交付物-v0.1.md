## 本文件是什么（核心定义，防止脱离索引被误读）

- **交付物 ≠ 字段 schema**。交付物是一份"**玩法库 + 元素库 + 红线**"，是 AI 生产这种内容时可调用的**表达资源包**。
- **模板 ≠ 文字填空模板**。每张玩法卡里的「30 秒分镜参考」就是模板——**拍摄者照着能拍出来的执行卡片**。
- **AI 生产时不是"填字段"**，而是"**从玩法库里选一张卡片并展开**"。

> 定义来自用户 2026-04-13 裁决。详见 [_index.md](_index.md) §0。

## A. 北极星

**观众看完会觉得"原来普通生活不是没内容，是没人会这样看"——产生"这就是我的生活"式共鸣，情绪被轻轻托住。**

> **三份原话备查：**
> - GPT5.4：会觉得"原来普通生活不是没内容，是没人会这样看"
> - 深度研究：产生"这就是我的生活"式共鸣，情绪被轻轻托住
> - compass：从"好看画面堆砌"走向声音景观和时间胶囊

## B. 元素库（原子层）

### B1 视觉元素
- 固定机位（同一角落重复拍）
- 物件痕迹（椅背外套 / 耳环 / 纸袋 / 钥匙 / 衬衫滴水）
- 极近景（手 / 衣角 / 门缝光）
- 超微距 / 地面视角（让熟悉变陌生）
- 装置艺术式陈列（食物残渣/一周衣服叠成山）
- 照片 dump（刻意保留瑕疵 / 糊 / 偏）
- 轮播卡片（不动画）

### B2 听觉元素
- 六声 / 十声蒙太奇（日常声音）
- 近场收音
- 静音断点（完全静音的空镜）
- 窗外天空长镜头 + 仅音频
- 无 BGM（声音本身是内容）
- 环境音 + 极弱 BGM

### B3 人物元素
- 不露脸（只有痕迹）
- 独人 + 物件
- 创作者手部特写
- 空房间作为"主角"

### B4 场景元素
- 同一角落（门口 / 椅子 / 试衣凳 / 柜台）
- 家 / 自家店（默认）
- 街角 / 地铁 / 公园（路过空间）
- 反预期角度（地面视角 / 超微距）

### B5 情绪元素
- 被理解感（不是被展示）
- 私密 > 公开
- 隐形线索（碎片之间有不言自明的主题）
- 体感优先 / 时间优先

### B6 文字元素
- 极少字（一句感受 / 一个时间戳）
- 完全无字
- 无解释 / 无事件说明
- 字幕只写心情的 1 个词

### B7 时间元素
- 同一时刻 5-7 天（固定机位+时间褶皱）
- 季节线索
- 时间对话（今天 vs 童年）
- 六声一日（声音编号 1-6）
- 同一椅子四次

### B8 反差元素
- 固定空间 vs 人物微差
- 视觉缺失 vs 声音丰富
- 日常 vs 微距陌生感
- 现在 vs 过去（双重时间线）

### B9 隐喻元素
- 痕迹 > 出镜（衣角/外套/钥匙 作为"人存在过的证据"）
- 声音 > 画面
- 椅子作为"一天的容器"
- 衣服作为"时间的标点"

## D. 服装行业垂直机会

日常碎片是服装行业**被严重低估的类型**。服装本来就会留下日常使用痕迹，而大多数品牌只拍"衣服新上身"那一刻，很少拍"衣服被一天真实使用之后"的样子。后者更像生活，也更像信任。

### 三个未被开发的角度

1. **"把衣服当作时间的标点"**——外套挂着、衣角露出、袖口被卷起，这些比镜头怼脸更真实。C3-1 / C3-2 / C3-5 都是直接入口
2. **"声音媒介"（C3-4 六声一日）**——服装的摩擦声、衣架声、纽扣声是独占的"日常声音资产"
3. **"陪伴感碎片"（C3-8 双重时间线）**——家里人穿过的老衣服与今天你穿的衣服对照——是品牌最柔软的叙事

### 行业趋势支撑
- 平台侧强调"真实利他"的内容逻辑——碎片只要能让人感到被理解，就天然有传播力
- 小红书 playbook 对"生活分享 / 日常 Vlog"的内容形态线索

## F. AI 辅助边界

| 环节 | AI 能做 | AI 不能做 |
|---|---|---|
| 选题 | 固定时刻建议 / 痕迹清单 / 感官分类 | 替代"观察到什么" |
| 脚本 | 声音清单 / 排序逻辑 / 对照主题 | 替代选择的直觉 |
| 拍摄 | 固定机位建议 / 近景构图 | 替代现场收音 |
| 剪辑 | 节奏草案 / 声画对位建议 | 替代"灵魂的选择逻辑" |
| 发布 | 标题 / 标签 | 替代共鸣 |

> **F 核心判断：** 碎片有没有灵魂，全看创作者有没有观察到"微小但有意味"的瞬间——AI 可帮做镜头分组、语气删减、节奏建议，但"选择逻辑"必须由人决定，否则必然模板化（GPT5.4 + 深度研究）。

## 与试点版（personal_vlog）的差异点

1. **核心差别：连续性**。personal_vlog 有"一天的连续感"，daily_fragment 是"不连续的瞬间集合"——即使用相似元素（痕迹、不露脸），组织方式完全不同
2. **突破性玩法数量多（8 张）：** 碎片类型对"选择逻辑"的需求最大，所以突破点也最多
3. **最强调"声音"：** C3-4 六声一日、C3-7 诗歌朗诵式碎片——声音在本 ContentType 里几乎是主角，比 personal_vlog 更突出
4. **AI 辅助边界最窄：** 碎片的灵魂几乎完全靠"选择逻辑"，AI 只能做工具辅助
