## 本文件是什么（核心定义，防止脱离索引被误读）

- **交付物 ≠ 字段 schema**。交付物是一份"**玩法库 + 元素库 + 红线**"，是 AI 生产这种内容时可调用的**表达资源包**。
- **模板 ≠ 文字填空模板**。每张玩法卡里的「30 秒分镜参考」就是模板——**拍摄者照着能拍出来的执行卡片**，含画面 + 声音 + 动作 + 节奏。
- **AI 生产时不是"填字段"**，而是"**从玩法库里选一张卡片并展开**"。

> 定义来自用户 2026-04-13 裁决。详见 [_index.md](_index.md) §0。

## 本文件与个人创作类 7 份交付物的关键差异

**企业叙事类 vs 个人创作类的核心区别：**

| 维度 | 个人创作类（batch 1） | 企业叙事类（batch 2，本文件所在） |
|---|---|---|
| **创作者身份** | 素人 / 个人博主 / 店员个体 | **企业主 / 主理人 / 创始人 / 老板身份** |
| **出镜动机** | 生活分享 / 自我表达 | **判断外化 / 代价显形 / 信任预支** |
| **观众期待** | "像我 / 陪我 / 逗我" | **"可信 / 有判断 / 有性格 / 敢承担"** |
| **服装角色** | 时间容器 / 情绪皮肤 / 身份接口 | **决策对象 / 证物 / 代价痕迹 / 人格外化** |
| **实拍可行性** | 1 人 + 1 手机 + 真实场景 + 200 元 + 4 小时 | **同 + 允许 1-2 同事临时帮忙 + 允许企业自有空间**（event_documentary / process_trace 可放宽到 1-3 人小团队）|
| **最危险的红线** | "AI 味 / 精致到假" | **"像广告 / 像客服 / 像精英励志片 / 像 PPT 口播"** |

**对 founder_ip 的具体含义：** 老板露脸不难，**难的是"不像老板在卖货"**。所有玩法的共同目标是让观众**先信这个人，再信他的牌子**——而不是反过来。

## A. 北极星

**观众看完后，心里不是"这老板在卖货"，而是"这个人有判断、有代价、有性格，我开始相信这家公司是由一个真实的人在做真实的事，我愿意把信任先预支给他、再预支给他的牌子"。**

> **三份原话备查：**
> - GPT5.4：不是"这老板在卖货"，而是"这个人有判断、有代价、有性格，我愿意先信他，再信他的牌子"
> - 深度研究：我开始相信这家公司是由一个真实的人在做真实的事，愿意把信任预支给品牌（而不是被"成功学话术"劝服）
> - compass：从"大家好我是 XX 创始人"到沉默叙事——算法和观众都在奖励"有灵魂的真实"而非"有套路的表演性真诚"

## B. 元素库（原子层）

### B1 视觉元素
- 手持跟拍 / 观察式镜头（非正面对镜）
- 手部特写（摸布料 / 拉版型 / 翻样衣 / 系扣子）
- 物件特写（旧样衣 / 退货单 / 改版纸条 / 纽扣 / 钥匙 / 面料小样）
- 桌面俯拍固定机位（"证物剧场"核心构图）
- 门口 / 门槛单机位（"身份切换"核心构图）
- 侧脸 / 背影 / 半张脸（不完整露脸）
- 反差场景（菜市场 / 美术馆 / 暴雨 / 极限环境）
- 工坊 / 打版台 / 收银台 / 仓库（企业自有空间）

### B2 听觉元素
- 布料摩擦声 / 拉链 / 衣架撞桌
- 门锁 / 钥匙串 / 脚步
- 剪刀 / 缝纫机 / 针线声
- 真实现场同期声（手机原声收录）
- 完全无 BGM 或极弱 BGM
- 保留口误 / 停顿 / 未完整的语音
- 半句语音消息 / 未发送的聊天记录声音

### B3 人物元素
- 创始人本人（默认）
- 创始人 + 身边关系人（家人 / 老员工 / 导购 / 同事）
- 1-2 同事临时帮忙拍摄（允许，符合企业叙事放宽条款）
- 不露脸 / 只露手 / 只露背影
- 跨代际同框（父母 / 伴侣 / 老员工）

### B4 场景元素
- 办公室（允许但要反套路用）
- 工作室 / 打版台 / 样衣间
- 店面 / 柜台 / 试衣间
- 仓库 / 车间 / 后仓
- 极限场景（珠峰 / 牧场 / 棉田 / 供应链现场）
- 反差场景（日常错位：菜市场 / 美术馆 / 暴雨）
- 门口 / 门槛（"进出企业身份"的物理隐喻）

### B5 情绪元素
- 判断压力（"今天必须做一个选择"）
- 代价显形（"这让我晚上睡不着"）
- 局限承认（"我也会被焦虑动摇"）
- 真实犹豫（不立刻给结论）
- 羞耻感 / 自我审视（不回避失败）
- 倔强（"还是决定这样做"）
- 非励志（避开"继续加油"式收尾）

### B6 文字元素
- 极少字幕 / 单句金句
- 手写改版纸条 / 退货单作为"证据字幕"
- 一句反鸡汤金句（例："这个牌子不是从梦想开始的，是从一次次不好意思拿出来给人看的东西开始的"）
- 无结论字幕 / 无 CTA
- 避免"关注我 / 私信我"类营销话术

### B7 时间元素
- 单一决策瞬间（不讲完整创业史）
- 桌面物件序列（时间不是线性的，是物件的）
- 门口 30 秒角色切换（早 / 中 / 晚 / 夜 同一门口）
- 静默断点（2-3 秒凝视 / 停手）
- 留白收尾（不给结论）

### B8 反差元素
- 柔软人设 vs 粗粝决策
- 成功 vs 失败样衣
- 办公室道理 vs 现场犹豫
- 晚礼服 × 菜市场（场景反差）
- 代际碰撞（经验 vs 直觉 / 稳妥 vs 冒险）
- "我很正确" vs "我也会犯错"

### B9 隐喻元素
- 旧样衣 = 人格化错误档案 / 老板过去版本的尸检报告
- 桌面物件 = 责任与压力的具象化
- 门口 = 身份开关 / 进出企业角色的物理隐喻
- 面料 = 价值观载体
- 服装 = "老板和世界交手的媒介"
- 代际对话 = 价值观翻译器

## C. 玩法库（核心交付物）

### C1 主流玩法（默认禁用 / 低优先）

> 本档玩法在服装行业 founder_ip 场景下**危害尤其大**——不仅无效，还会反向伤害品牌（定价敏感 + 精英感警惕）。笛语平台对 founder_ip 的 C1 档应该**强默认禁用**，比个人创作类更严格。

#### 玩法卡 #C1-1 —— "办公室对镜鸡汤口播"

| 字段 | 内容 |
|---|---|
| 档位 | 🚫 主流 / 强默认禁用 |
| 一句话定义 | 老板坐在办公室 / 车里 / 会议桌前对着镜头讲创业初心、行业认知、品牌理念、做产品有多难。画面通常是正面半身，字幕密密麻麻，BGM 是"励志轻燃" |
| 北极星映射 | ❌ 观众接收到的是"结论"，不是"这个人"；像 PPT 口播，AI 最容易模仿——在"老板 IP 很卷"的语境下，**单纯口播极容易被归类为营销话术** |
| 表达元素清单 | B1 正面中景 / B2 口播 / B6 密字幕 / B2 轻励志 BGM / B4 办公室/车内 / B5 结论先行 |
| 服装行业钩子 | ❌ 服装沦为"老板顺手穿的品牌衣服"，不是叙事主体，存在感很弱 |
| 实拍可行性 | 🟢 易拍，但**门槛低 → 同质化 → 反向伤害** |
| 30 秒分镜参考 | （主流玩法不写分镜，避免被复用） |
| AI 辅助度 | ⚠️ AI 最容易模仿，也因此最容易被识别为"AI 生成的假口播" |
| 来源 | GPT5.4 §1 + 深度研究 §founder_ip + compass §一（三份都点名的主流陷阱） |
| 备注 | **服装行业特别警告：** 一旦观众产生"包装感"，会反向质疑定价与品质 |

### C2 进阶玩法（少数高手在用）

#### 玩法卡 #C2-1 —— "决策日记 / 决策现场纪录"

| 字段 | 内容 |
|---|---|
| 档位 | ⭐ 进阶 |
| 一句话定义 | 不拍"成功学"，只拍今天必须做的**一个决定**：这批布要不要退？这个颜色要不要砍？这批样衣退货处理？内容不追求"完整创业史"，而是压缩成一个**真实决策现场**。老板只说必要的 1-2 句（甚至不说），镜头跟着手和物走 |
| 北极星映射 | ✅ 把"老板内容"从观点输出升级成**判断过程展示**——价值观不是说出来的，而是通过取舍自己显形 |
| 表达元素清单 | B1 手持跟拍 / B1 手部与物的特写 / B4 样衣/面料/报价单 / B2 现场同期声（布料摩擦/剪刀/衣架）/ B5 停顿犹豫 / B6 短句解释"为什么" / B5 节奏断点 |
| 30 秒分镜参考 | **0-5 秒：** 手持镜头跟随老板走到打版台 + 桌上摊着两块不同颜色的布样 \| **5-15 秒：** 老板的手摸两块布，沉默 5 秒 + 只说一句"这个颜色太安全" \| **15-22 秒：** 他把较保守的一块推到一边，拿起大胆的那块 + 字幕"今天只做一个选择" \| **22-30 秒：** 镜头停在被推开的那块布上 2 秒，不给结论 |
| 服装行业钩子 | ⭐⭐⭐ 服装是"决策对象"——每一处版型、颜色、面料都是老板人格的外化 ; 别的行业未必有这么具象的决策证据 |
| 实拍可行性 | 🟢 1 人口播或 1 位同事拿手机即可 / 预算几乎为 0 / 不依赖专业演员或专业摄影 |
| AI 辅助度 | ✅ 能做：决策清单建议 / 镜头顺序 / 字幕润色 ; ❌ 不能做：替代真实犹豫的节奏 |
| 来源 | GPT5.4 §1 进阶玩法 1 + 深度研究 §founder_ip 进阶玩法 1（合并） |
| 备注 | 信息密度来自"现场问题 + 决策过程"，可验证、不可套模板——"founder-led marketing" 之所以有效，不在于老板露脸本身，而在于创始人公开分享过程 |

#### 玩法卡 #C2-2 —— "自我矛盾暴露 / 反向透明（讲一次我做错了什么）"

| 字段 | 内容 |
|---|---|
| 档位 | ⭐ 进阶 |
| 一句话定义 | 不塑造"无所不能老板"，而塑造"有局限但还在扛的人"。选一个可公开的小失败：某批次起球 / 某版型不适合更多身材 / 某次活动服务没跟上。结构：失败现场（证据）→ 你当时怎么判断（人性）→ 你怎么补救（行动）→ 现在的结果（验证） |
| 北极星映射 | ✅ 观众更容易信"有裂缝的人"而不是"永远正确的人"——裂缝让人格成立 |
| 表达元素清单 | B1 近景脸部 / B5 口误和停顿保留 / B1 失败样衣 / B6 未发送的聊天记录 / B1 返工现场 / B5 轻度自嘲 / B2 非励志配乐甚至不用 BGM |
| 30 秒分镜参考 | **0-5 秒：** 桌上摊着一件起球的样衣 + 字幕"这批我翻车了" \| **5-15 秒：** 老板的手指捏起起球的地方 + 低声"当时是我选错了面料供应商" \| **15-22 秒：** 镜头切到他重新打电话约新供应商的现场 \| **22-30 秒：** 镜头停在新样衣的领口特写 + 字幕"这次不敢保证不会再错，但至少我没假装没发生过" |
| 服装行业钩子 | ⭐⭐⭐ 服装在这里不是"成功商品"，而是"错误、挣扎、判断的痕迹板"——服装行业更容易因定价/材质引发质疑与舆情，因此"透明型创始人叙事"既是**增长工具**，也是**风险对冲工具** |
| 实拍可行性 | 🟢 固定机位就能拍 / 剪映加字幕即可 / 预算 0-30 元 / 需把握边界（不泄露敏感商业机密） |
| AI 辅助度 | ✅ 能做：失败案例结构建议 ; ❌ 不能做：替代"敢讲瑕疵"的真诚 |

### C3 突破性玩法（笛语平台主推）

> 本档是笛语平台对 founder_ip 的核心交付物——**服装行业 founder 的"人格化表达"空间远超其他行业**，因为服装天然有可拍的"判断证据链"。AI 生产 founder_ip 时应该**优先**从 C3 取卡。

## D. 服装行业垂直机会

服装行业做 `founder_ip` 的**最大优势**不是"老板能穿自家衣服"，而是**老板所有判断都能被看见**：面料怎么摸、版型怎么拉、颜色怎么犹豫、旧样衣怎么舍不得扔。

### 三个未被开发的角度

1. **"身体化判断的人"**（C3-1 旧样衣 / C3-2 不看镜头一天 / C3-4 桌面证物）——他/她对衣服的下意识动作，本身就是人格内容。别的行业的 founder 内容很容易停在抽象理念，服装行业却天然拥有可拍的**证据链**
2. **"透明型创始人叙事作为风险对冲工具"**（C2-2 反向透明）——服装行业容易因定价/材质引发质疑与舆情，"承认做错"既是**增长工具**，也是**风险对冲**
3. **"一人一手机降维"的方法论背书**（C3-6 一镜到底工坊漫游）—— 已有国际级品牌亲自证明"一人一手机也能拍奢侈品级内容"，中小品牌没有任何技术借口（v0.2 M4 无痕处理：删去具体品牌名）

### 服装行业 founder_ip 方向（v0.2 M4 无痕处理：删去全部具体标杆案例姓名与数据 / 仅保留方向描述）
- **极限场景人格化方向** —— 创始人把自家产品推进高海拔 / 原料产地 / 特殊工况做实测 / 真实身体反应就是证言
- **生活方式沉浸方向** —— 把穿搭融入旅行 / 街头 / 日常 / 让社交证明代替自我推销
- **街头偶遇验证方向** —— 用陌生路人认出的那一刻作为品牌被记住的证据
- **一镜到底工坊漫游方向** —— 手机一镜到底拍工坊日常 / 一句话收尾
- **价值观先行方向** —— 把品牌主张放在产品前面 / 用立场筛选目标用户群
- **个人身份前置方向** —— 把创始人个人经历作为品牌长期叙事的主角
- **学术研究验证方向** —— "个人品牌承接企业形象"作为时尚 CEO 的通用路径

### 行业趋势支撑
- "老板下场做 IP"已被总结为近年显著趋势
- TikTok 2026 报告 "authenticity > polish" 对创始人内容尤其适用
- 服装行业观众对"完美人设"天然警惕——因此 C2-2 反向透明和 C3-1 旧样衣审判庭是**服装行业的特殊红利**

## F. AI 辅助边界

| 环节 | AI 能做 | AI 不能做 |
|---|---|---|
| 选题 | 今日决策清单（3 选 1）/ 物件重量建议 / 反差场景清单 | 替代"你今天真的在意什么" |
| 脚本 | 镜头顺序 / 口播收缩 / 字幕润色 / 脱口秀三段结构（设置→反转→收束）| 替代创始人真实情绪 |
| 拍摄 | 镜头清单 / 走位路线 / 物件顺序 | 替代真实现场的偶然性 |
| 剪辑 | 节奏点 / BGM 候选 / 字幕极简化 | 决定"删掉哪些解释、保留哪些犹豫" |
| 发布 | 评论区高频问题提炼 / 标题 / 标签 | 替代"这个人是谁"的人格感 |

> **F 核心判断（GPT5.4 + 深度研究 合并）：** **可深度辅助**——能做选题、结构、口播收缩、字幕润色、镜头清单、BGM 候选、评论区高频问题提炼 ; 但**"人味证据"必须来自老板真实反应，AI 不能代演**。**不建议生成大段口播**（最容易 AI 味）。
