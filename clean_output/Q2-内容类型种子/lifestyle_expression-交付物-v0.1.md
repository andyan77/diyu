## 本文件是什么（核心定义，防止脱离索引被误读）

- **交付物 ≠ 字段 schema**。交付物是一份"**玩法库 + 元素库 + 红线**"，是 AI 生产这种内容时可调用的**表达资源包**。
- **模板 ≠ 文字填空模板**。每张玩法卡里的「30 秒分镜参考」就是模板——**拍摄者照着能拍出来的执行卡片**，含画面 + 声音 + 动作 + 节奏，不是"标题 + 正文 + 标签"式的字段拼装。
- **AI 生产时不是"填字段"**，而是"**从玩法库里选一张卡片并展开**"。

> 这个定义来自用户 2026-04-13 裁决，是本目录全部 v0.x 草案的共同前置。详见 [_index.md](_index.md) §0。**2026-04-10 曾因"用 schema 抽象覆盖业务灵魂"翻车，4 份过度工程化文档已归档，本批工作严格使用"玩法卡片化"模型，不退回字段拼装思维。**

## A. 北极星

**观众看完不是"我被教育了一种理念"，也不是"我被说服"，而是"我突然想靠近这种活法，并且感到被允许这样活"。**

> **三份原话备查：**
> - GPT5.4：不是"我被教育了一种理念"，而是"我突然想靠近这种活法"
> - 深度研究：不是"被说服"，而是"被带入一种节奏与价值观"，看完后对"这种活法"产生向往或被允许
> - compass：算法奖励"活人感"——"发疯""抽象"式真实比精致感更受欢迎；55% 用户更信任人类创作内容

## B. 元素库（原子层）

### B1 视觉元素
- 色温分区（冷蓝 / 暖黄 / 顶白 / 路灯橙）
- 仪式动作特写（叠衣 / 熨烫 / 熄灯 / 放杯子）
- 物件痕迹（磨损的袖口 / 反复缝补的线 / 旧牛仔裤褪色）
- 房间不同空间作为叙事地图
- 手工书写 / 纸笔 / 信封

### B2 听觉元素
- 环境音主导（雨声 / 咖啡机 / 翻书 / 水壶沸 / 针线 / 衣料摩擦）
- 近场收音（布料摩擦 / 拉链 / 扣子）
- 完全无人声 / 低声自语
- 几乎无 BGM 或极弱 BGM

### B3 人物元素
- 独人（不露脸或只露手部）
- 双人关系映照（跨代际 / 跨性别 / 习惯差异）
- 无"主张句讲述者"人设

### B4 场景元素
- 自家卧室 / 厨房 / 玄关（默认）
- 反预期场景（店后仓 / 地铁站长椅 / 凌晨便利店 / 雨后街边）
- 四空间分区（卧室 / 店 / 路灯街边 / 夜里厨房）
- 咖啡馆 / 公园 / 地铁站（真实公共空间）

### B5 情绪元素
- 被允许感（不逞强、不表演）
- 筛选感 / 边界感 / 秩序感
- 高级 = 陪伴 ≠ 更新
- 不主张、不总结、不收尾成 slogan

### B6 文字元素
- 极少字幕 / 只一句
- 手写信件作为旁白文本
- 完全无字幕（纯沉浸）
- 关键词字幕（3 个词，不写完整句）

### B7 时间元素
- 按空间/色温排序而非时间线性
- 仪式动作重复结构
- 散文式意识流（跳跃剪辑）
- "被美化的一天"vs"真实的一天"同一天双版本

### B8 反差元素
- 柔软穿着 × 粗粝场景
- 精致画面 × 真实混乱
- 外部秩序 × 内部波动
- 理想一天 × 真实一天

### B9 隐喻元素
- 衣物作为"价值观筛子"（舒适优先 / 场合优先 / 心情优先）
- 衣物作为"情绪护城河"
- 旧衣作为"陪伴的证据"（反消费主义高级感）
- 房间空间作为"生活宇宙"

## D. 服装行业垂直机会

生活方式内容到了服装行业，最容易犯的错是把"生活方式"偷换成"品味摆拍"。真正有空间的方向，是**让服装变成一种立场**：舒适优先、行动优先、情绪优先、关系优先、身体优先。

### 三个未被开发的角度

1. **"把一套衣服和一种真实活法绑定"**——TikTok 2026 强调 "emotional ROI"，观众越来越在意"为什么值得买"而不是只看便宜不便宜。服装品牌如果能把一套衣服和一种真实活法绑定，比单纯的审美输出更有穿透力（GPT5.4 引）
2. **"把衣服当作价值观的行为接口"**——你怎么叠衣服、怎么修补、怎么在不同场景穿同一件基础款，都是比"口播主张"更可信的生活态度输出（深度研究 引）
3. **"以旧为高级"**——反消费主义高级感是下一个窗口 / 降维到个人尺度的"旧物修补 / 长期使用 / 痕迹审美"叙事，C3-5 是直接对接口（v0.2 M4 无痕处理：删去原引用品牌名）

### 行业趋势支撑
- TikTok 2026 "polished 内容正被 grounding 内容取代"（compass 引）
- 2024 小红书关键词"活人感"——"发疯""抽象"式真实比精致感更受欢迎（compass 引）
- 小红书 playbook 对"生活分享 / 日常 Vlog / 上身讲解"内容形态线索（深度研究 引）

## 与试点版（personal_vlog）的差异点

1. **元素库维度不同：** personal_vlog 侧重"时间/痕迹/见证者"，lifestyle_expression 侧重"色温/仪式/反差场景"。B 库字段完全不同
2. **北极星更强调"允许感"：** personal_vlog 是"陪我一段"，这里是"我被允许这样活"——前者是陪伴，后者是解放
3. **AI 辅助边界更窄：** 这个 ContentType 更依赖"真立场"，AI 能辅助的环节比 personal_vlog 少一些
4. **服装钩子更接近"价值观载体"：** 比 personal_vlog 的"时间容器"更抽象一层，反而离品牌长期资产更近
