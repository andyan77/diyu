## 本文件是什么（核心定义）

- **交付物 ≠ 字段 schema**。交付物是一份"**玩法库 + 元素库 + 红线**"，是 AI 生产这种内容时可调用的**表达资源包**。
- **模板 ≠ 文字填空模板**。每张玩法卡里的「30 秒分镜参考」就是模板——**拍摄者照着能拍出来的执行卡片**。
- **AI 生产时不是"填字段"**，而是"**从玩法库里选一张卡片并展开**"。

> 定义来自用户 2026-04-13 裁决。详见 [_index.md](_index.md) §0。

## 本文件的"1-3 人小团队"放宽条款

**process_trace 是企业叙事类 6 份里**第 2 个使用"1-3 人小团队"放宽条款的文件**（event_documentary 是第 1 个）。放宽原因：

1. **工艺是协作的** —— 一件衣服从面料到成品经过多人的手，单人镜头无法完整记录"不同的手在同一件物品上留下的判断"
2. **车间是跨空间的** —— 打版台 / 裁剪 / 缝纫 / 熨烫 / QC 分布在不同工位，需要至少 1-2 人协同拍摄才能跟上节奏
3. **不可逆动作需要同期佐证** —— 下剪 / 压线 / 洗水前的定格动作，需要有人同步记录环境和操作者表情

**但仍严守的边界（与 event_documentary 完全一致）：**
- ❌ 不允许专业摄制组 / 航拍 / 多机位导播
- ❌ 不允许专业灯光 / 专业收音麦克风阵列
- ❌ 不允许雇佣专业演员 / 舞者代替真实工匠（本 ContentType 的灵魂是"**真实工匠的手**"）
- ✅ 最多 1-3 人同伴 + 手机/微单 + 现场自然光 + 最多 1 个领夹麦
- ✅ 后期仅用剪映 / CapCut 等手机端工具
- ✅ 单条成品 ≤ 4 小时；但允许**素材采集跨天**（见 C2-4 全链路、C3-8 一根线的旅程等长期积累型玩法）

**自检三问**（每张突破性玩法卡都必须答）：
1. 1-3 人小团队用手机能拍出来吗？
2. 是否依赖任何"专业身份"（专业摄影师 / 调色师 / 专业演员 / 专业收音）？
3. 单条成本能控制在 200 元以内吗？（原料产地访问允许必要交通费，但不包括差旅团队）

**任一不达标 → 玩法降级或重写。**

## A. 北极星

**观众看完不只是"知道工艺步骤"，而是"我愿意相信你没有骗我"——对"这件衣服为什么值得被认真对待"产生身体层面的理解，甚至觉得：原来品质不是一句话，是一连串不能偷懒的动作。**

> **三份原话备查：**
> - GPT5.4：不只是"知道工艺步骤"，而是对"这件衣服为什么值得被认真对待"产生身体层面的理解
> - 深度研究："**我愿意相信你没有骗我**"——对材质、工艺与生产伦理产生基本信任
> - compass：从"走进工厂"到"无声感官沉浸"——服装溯源最缺的就是"**人**"和"**可核验的证据**"

## B. 元素库（原子层）

### B1 视觉元素
- 手部特写（不同人的手：年轻 / 粗糙 / 戴戒指 / 老茧）
- 极微距（面料纤维 / 走线 / 染色渗透）
- 对花 / 回弹 / 缝线收口的近景
- "不可逆动作前的静止"（下剪前 / 压线前 / 洗水前）
- 被拦下的成衣（瑕疵特写）
- 证据链（原料标签 / 入库单 / 抽检 / 成品批次）
- 夜班低光 / 自然光
- 车间 / 打版台 / 熨烫台 / QC 台
- 一根线的旅程（棉花田 → 纺纱 → 织布 → 染色 → 裁剪 → 缝制 → 开箱）

### B2 听觉元素
- 面料拉伸声 / 蒸汽压过 / 布料滑行速度
- 剪刀阻力感 / 缝纫机节奏
- 机器停机的静默
- 布料摩擦 / 拉链 / 纽扣
- 呼吸声 / 停顿
- 纸张翻动 / 订书机 / 铅笔
- 扫码"滴" / 机器启停
- 无 BGM 或极弱

### B3 人物元素
- 不露脸（只拍手）
- 不同人的不同手
- 夜班工人与机器共处
- QC 员只说"一毫米偏了"
- 老裁缝（如 VOA 丝绸 70 多岁老裁缝）
- 非遗传承人（如"朱伯伯的苏罗" / 江寻千）

### B4 场景元素
- 自家 / 合作小工厂的车间（允许进入）
- 原料产地（棉花田 / 牧场 / 纺纱厂——如 compass 引的藏里羊）
- 打版台 / 裁床 / QC 台
- 质检区
- 夜班车间（低光 / 安静）
- 桌面实验区（对比测试）

### B5 情绪元素
- 敬畏感（"这一刀下去就不能回头了"）
- 风险感 / 专业判断
- 克制（不煽情）
- "举证"式的节奏
- 非遗传承的安静尊严

### B6 文字元素
- 证据标签（"这能证明什么"）
- 一句话结论（"3 毫米的差别，30% 的价格差"）
- 完全无字 / 极简字幕
- 标注敏感信息打码
- 避免"严格把控"类空话

### B7 时间元素
- 单一工序深挖（不贪完整）
- 静止 + 动作（前一秒的停顿）
- 一根线的完整旅程（从原料到开箱）
- 夜班的慢节奏
- 对比实验的前后时间

### B8 反差元素
- 被放行 vs 被拦下
- 机器动 vs 机器停
- 流程展示 vs 真实风险
- 合格品 vs 瑕疵品
- 手的力量 vs 机器的力量
- 大量全流程 vs 一个难以作假的瞬间

### B9 隐喻元素
- 手 = 品质在负责
- 一刀 = 不可逆的专业判断
- 被拦下的衣服 = 品牌的边界感（知道什么不该被卖出去）
- 微距纹理 = 一块布是一个微观宇宙
- 一根线的旅程 = 供应链不是抽象词
- 侦探找指纹 = 可核验的推理

### C2 进阶玩法（少数高手在用）

#### 玩法卡 #C2-1 —— "只拍一个最难伪装的步骤"

| 字段 | 内容 |
|---|---|
| 档位 | ⭐ 进阶 |
| 一句话定义 | 不试图拍全流程，而只拍一个**最容易看出真假的节点**：比如裁剪对花有没有对齐、缝线收口是否干净、面料回弹是否真实、整烫后肩线有没有立起来、QC 为什么把一件看似没问题的衣服拦下 |
| 北极星映射 | ✅ **不贪完整，而抓住"最难作假的一刀"**——观众会更容易信 |
| 表达元素清单 | B1 单一工序深拍 / B1 局部特写 / B1 慢动作只用于关键点 / B2 自然机器声 / B6 少量解释 |

| 服装行业钩子 | ⭐⭐⭐⭐ **服装行业最适合做"难伪装步骤"**——对花 / 缝线 / 回弹 / 肩线 / 收口都是品质差异集中爆发的节点 |

| AI 辅助度 | ✅ 能做：难伪装节点清单 / 镜头建议 ; ❌ 不能做：替代一线人员的经验判断 |

#### 玩法卡 #C2-3 —— "证据链溯源 / 我只展示 5 个能核验的点"

| 字段 | 内容 |
|---|---|
| 档位 | ⭐ 进阶 |
| 一句话定义 | 不拍"全流程"，只拍 5 个"**证据点**"：原料标签、入库单/检验报告（关键信息打码）、抽检动作、工序关键节点、成品批次标识。每个点只给一句"**这能证明什么**" |

## D. 服装行业垂直机会

服装行业做 `process_trace` 的**独特优势**，是**工艺差异能够被肉眼、耳朵、手感联想同时感知**。你不是在讲一个抽象供应链，而是在讲：**为什么这件衣服能站起来、能垂下去、能久穿不散、能摸起来不像便宜货**。

### 三个未被开发的角度

1. **"一个难以作假的瞬间"（C2-1 + C3-2 + C3-4）** —— 最有爆相的方向不是全流程旅游片，而是对花、回弹、拦货、不可逆动作前的停顿——**单点深拍比全流程更有说服力**
2. **"感官溯源"（C3-6 ASMR + C3-7 微距）** —— 把工艺从知识变成**视觉 + 听觉 + 触感联想**的混合体
3. **"人与机器共处"（C3-5 夜班）** —— 服装溯源最缺的就是"**人**"——夜班镜头天然把"人"拉回画面中心

### 服装行业 process_trace 方向（v0.2 M4 无痕处理：删去全部具体账号 / 品牌 / 数据 / 仅保留方向描述）
- **夫妻老板直播输出工艺干货方向** —— 用直播间作为长期内容产线
- **顶级面料溯源方向** —— 去产地 / 走进工厂的实地拍摄
- **非遗织造传承人方向** —— 用短视频让传统工坊重获生命力
- **非遗手艺学习者方向** —— 跨技艺学习（铁花 / 糖画 / 织绣等）
- **老裁缝失误瞬间方向** —— 暴露不完美比展示完美更打动人
- **制造透明内容海外账号方向** —— "Made in / Behind / Material" 系列叙事
- **品牌透明制造叙事方向** —— "Meet My Material / 看见原料" 类品牌内容
- **听觉时装视频方向** —— 早期 ASMR 时装视频已开创该路径
- **感官沉浸品牌叙事方向** —— "I Am Your Mirror" 类高级感叙事

### 行业趋势支撑
- 抖音非遗项目覆盖率 **99.74%** / 相关视频播放总数 **3726 亿** / 获赞 94 亿
- 抖音新中式服装月销 **5 亿** / 马面裙订单量同比增长 **841%**
- TikTok 服装工厂相关标签参与**超千亿**
- Gen Z 对 fashion factories 的兴趣在上升（Vogue 2024）
- **"knowing is the new owning"**

## F. AI 辅助边界

| 环节 | AI 能做 | AI 不能做 |
|---|---|---|
| 选题 | 工序拆解 / 难伪装节点清单 / 证据点候选 | 替代"哪里最能证明品质" |
| 脚本 | 镜头顺序 / 知识点转译 / 专业术语转人话 / 极简字幕 | 决定"**披露边界**" |

| 发布 | 标题 / 标签 / 封面 | 替代"品牌边界感" |

> **F 核心判断（GPT5.4 + 深度研究 合并）：** **中高辅助** —— 可辅助工序拆解、镜头顺序、知识点转译、字幕简化 ; 但"**哪里最能证明品质**"和"**披露边界**"必须一线人员/人为判断，AI 不能替代。
