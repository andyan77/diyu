## 本文件是什么（核心定义）

- **交付物 ≠ 字段 schema**。交付物是一份"**玩法库 + 元素库 + 红线**"，是 AI 生产这种内容时可调用的**表达资源包**。
- **模板 ≠ 文字填空模板**。每张玩法卡里的「30 秒分镜参考」就是模板——**拍摄者照着能拍出来的执行卡片**。
- **AI 生产时不是"填字段"**，而是"**从玩法库里选一张卡片并展开**"。

> 定义来自用户 2026-04-13 裁决。详见 [_index.md](_index.md) §0。

## 与 `founder_ip` 的边界（来自用户 G5 裁决，2026-04-13）

**判断标准不是"老板是不是也有岗位"，而是"内容的第一叙事重心是谁"：**

| 第一叙事重心 | 归属 |
|---|---|
| **"这个老板/主理人这个人"**（判断、人格、代价、性格） | `founder_ip` |
| **"一个岗位角色如何工作、如何解决问题、如何代表品牌说话"** | `role_work_vlog`（本文件） |

**生活化例子：** 同样拍老板在改版台前看样衣——
- 观众记住的是"**这个老板真会犹豫、真会扛**" → `founder_ip`
- 观众记住的是"**版师/主理人这个岗位到底怎么改版**" → `role_work_vlog`

**对 role_work_vlog 的含义：** 观众要接住的是"**岗位的智力和手感**"，不是"这个人的人格"。

## A. 北极星

**观众看完不只是"知道这个岗位在干嘛"，而是对这个岗位产生尊重、对品牌产生体感信任，甚至开始觉得"原来一件衣服背后站着的是一群有手感和判断的人"。**

> **三份原话备查：**
> - GPT5.4：对这个岗位产生尊重、对品牌产生体感信任 ; "原来一件衣服背后站着的是一群有手感和判断的人"
> - 深度研究：这家公司里的人可信、可亲、专业 ; 愿意因为"人"而相信服务与产品
> - compass：从"设计师的一天"到"无声手部 ASMR"——**岗位价值靠判断与手感显形，不靠流水账**

## B. 元素库（原子层）

### B1 视觉元素
- 手部特写（缝纫机 / 裁剪 / 熨烫 / 挂装 / 扫码 / 打包）
- 胸前 POV 第一视角
- 物件接力（一件衣服在多双手里传递）
- 旧陈列图 / 旧照片（"我以前是这样干的"）
- 失误证物（挂错的衣服 / 被退回的包裹 / 起褶的面料）
- 单机位后台 / 打版台 / 收银台 / 仓库
- 俯拍工作台

### B2 听觉元素
- 剪吊牌"咔嚓" / 胶带封箱 / 蒸汽熨斗 / 衣架滑金属杆
- 扫码枪"滴" / 对讲机沙沙 / 抽屉 / 门帘
- 缝纫机踩线 / 剪刀 / 针线
- 现场同期声为主，不加 BGM
- 偶尔保留半句同事对话

### B3 人物元素
- 单岗位独人
- **岗位接力**（同一件衣服经过仓库员/熨烫工/陈列师/导购/收银/包装）
- 不露脸 / 只露手和背影
- 一人分饰（"我以前 vs 我现在"）
- 同事自然入镜

### B4 场景元素
- 打版台 / 样衣间 / 工作室（设计 / 打版岗位）
- 店面柜台 / 试衣间 / 陈列区（导购 / 陈列岗位）
- 仓库 / 后仓 / 打包台（仓储 / 物流岗位）
- 熨烫台 / 清洗区（QC 岗位）

### B5 情绪元素
- 岗位尊严（不降级为"体力展示"）
- 判断显形（"为什么我在第 10 秒就知道这位顾客不能先推爆款"）
- 微小成就感（解决 / 被感谢 / 小挫败）
- 反"可爱员工人设"（不撒娇 / 不卖萌）
- 反"我很忙"的表演感

### B6 文字元素
- 岗位字幕标注（"仓库" / "熨烫" / "陈列" / "导购"）
- 短句判断语（"你先别看颜色，先看肩这里"）
- **内心字幕**（外部沉默，字幕只写心里那句：如"她刚刚皱眉了，我得换个版型"）
- 无长段口播 / 无"培训话术"

### B7 时间元素
- 一件衣服的接力时间线（仓库 → 陈列 → 试穿 → 包装 → 出门）
- 一镜交接班（早班 → 晚班 → 店长复核 → 关灯）
- 时间压缩（重复动作的节奏感）

### B8 反差元素
- 岗位动作的"粗活" vs "细活"
- 流程劳动 vs 判断智力
- 完美展示 vs 失误保留
- 岗位话术 vs 内心真实判断
- 沉默劳动 vs 产品价值

### B9 隐喻元素
- 衣服 = 岗位能力的"测试物"
- 手 = 岗位判断的外化
- 接力 = 品牌不是 logo，是很多双手的协作
- 岗位动作 = 一份日常交响曲（剪吊牌 / 封箱 / 蒸汽 / 衣架 / 扫码）
- 失误 = 经验的证据

### C2 进阶玩法（少数高手在用）

#### 玩法卡 #C2-1 —— "岗位视角不是'我做了什么'，而是'我怎么判断'"

| 字段 | 内容 |
|---|---|
| 档位 | ⭐ 进阶 |
| 一句话定义 | 导购不是拍自己接待了多少客，而是拍"我为什么在第 10 秒就知道这位顾客不能先推爆款"；仓库员工不是说今天理了多少箱，而是说"哪三种包法最容易让一件衣服到顾客手里时不掉价"；陈列岗位不是展示美图，而是展示"为什么这一件必须低一格挂" |
| 北极星映射 | ✅ 把"岗位"从劳动展示升级为**判断展示**——这才让观众产生敬意 |
| 表达元素清单 | B1 POV 镜头 / B1 手部动作 / B3 微表情 / B1 对比示范 / B6 少量旁白 / B2 现场声 / B6 具体判断语句 |
| 30 秒分镜参考 | **0-5 秒：** POV 镜头——导购看见一位顾客走进来 \| **5-12 秒：** 字幕"第 10 秒：她在看什么？" + 镜头扫过顾客视线停留的区域 \| **12-22 秒：** 导购走过去，手没有直接拿爆款，而是先摸了摸顾客视线停留的那件 \| **22-30 秒：** 字幕"不是她不买，是你不能先递错那件" |
| 服装行业钩子 | ⭐⭐⭐ 服装在这里是"被判断的对象"——越具体的版型 / 尺码 / 垂感 / 挂法，越能把岗位价值拍出来 |
| 实拍可行性 | 🟢 手机胸前位/手持就能完成 / 预算 0-50 元 / 无需专业演员 |
| AI 辅助度 | ✅ 能做：判断点清单 / 问题提炼 ; ❌ 不能做：替代真实现场经验 |
| 来源 | GPT5.4 §2 进阶玩法 1（Vogue 2025 "员工是新影响者" + Loewe 工艺师 + Ganni 团队 + Foot Locker EGC） |
| 备注 | 与 `founder_ip` C2-1 决策日记的区别：决策日记是**老板**做品牌级判断，本玩法是**一线岗位**做日常微判断 |

#### 玩法卡 #C2-2 —— "员工生成内容 EGC / 让岗位成为叙事者"

| 字段 | 内容 |
|---|---|
| 档位 | ⭐ 进阶 |
| 一句话定义 | 不把员工当"执行者"，而是让员工用自己的语言/视角记录：我今天最难的是哪位顾客、最开心的是哪次帮人找到合适版型 |
| 北极星映射 | ✅ 员工视角在"可信度"与"真实感"上具有结构性优势——**"我的雇主/我的 CEO"在信任度上天然高于"泛化 CEO"** |
| 表达元素清单 | B1 第一人称/半第一人称镜头 / B5 真实语气（可轻吐槽但不抱怨）/ B2 服务现场同期声 / B5 情绪拐点（解决/被感谢/小挫败）/ B6 少字幕 |
| 30 秒分镜参考 | **0-5 秒：** 你穿着工牌走进店里，对镜头说"今天有个客人让我想了很久" \| **5-15 秒：** 镜头切到你帮那位客人找衣服的画面（可以是重演）\| **15-22 秒：** "她要的不是显瘦，是不勒" \| **22-30 秒：** 字幕"她最后买了一件比我推荐贵 20 的" |
| 服装行业钩子 | ⭐⭐ 服装是"解决问题的工具"（遮肉 / 不勒 / 不透 / 不闷），员工是把工具用好的人 |
| 实拍可行性 | 🟢 1 人可拍（固定机位+自述）或同事帮拍 / 预算 ≤ 200 / 不需专业演员 |
| AI 辅助度 | ✅ 能做：把员工口语整理成不油的字幕 / 自动挑选高情绪片段 ; ❌ 不能做：AI 生成"统一人设台词"（最容易 AI 味） |
| 来源 | 深度研究 §role_work_vlog 进阶玩法 1 |
| 备注 | 连锁品牌已开始用员工灵感驱动社媒活动 ; "员工不是素材，是创意来源" |

### C3 突破性玩法（笛语平台主推）

#### 玩法卡 #C3-1 —— "一件衣服在不同岗位手里（岗位接力赛）"

| 字段 | 内容 |
|---|---|
| 档位 | 🌟 突破性 |

## D. 服装行业垂直机会

`role_work_vlog` 在服装行业最大的机会，是把"岗位"从服务动作拍成**身体经验**：店员看一眼肩线就知道不对，仓库摸一下包装就知道会掉价，陈列一伸手就知道这件该高一格还是低一格。这种判断**天然可视、可比、可复现**。

### 三个未被开发的角度

1. **"岗位如何通过衣服显露智力"（C2-1 判断展示 / C3-1 一件衣服接力）** —— 服装是岗位智力的外化。现在大多数服装岗位内容还停在"上班生活记录"，真正没被开发的是"**岗位如何通过衣服显露智力**"
2. **"手感即品牌资产"（C3-3 手部 ASMR / C3-4 无声工作记录）** —— 服装行业天然富声音（缝纫机、剪刀、熨斗、衣架），**这些声音可以独占成为品牌听觉资产**
3. **"判断即服务"（C3-5 内心字幕）** —— 导购的"眼力"和"判断"是服装零售最难标准化的软技能，也是最能打动顾客的价值

### 服装行业岗位内容方向（v0.2 M4 无痕处理：删去具体姓名 / 账号 / 品牌 / 数据 / 仅保留方向描述）
- **版型小课堂方向** —— 版师/打版员用手边工具讲原型法 / 把"看似很专业"的知识做成"3 分钟看懂"
- **探厂 Vlog 方向** —— 拜访多家工厂 / 对比不同生产线 / 服务 B 端和尾货回收场景
- **小店起家方向** —— 从最小的门店空间出发做内容 / 强调"小店也能做大"的可复制性
- **一衣多穿基础款方向** —— 用最少的衣服做最多的表达 / 强调实用和普惠
- **品牌员工即代言方向** —— 从老板到社媒经理 / 让员工而非外部模特成为品牌形象代言人
- **EGC (Employee Generated Content) 策略方向** —— 员工出镜内容的 engagement 远超品牌统一制作的内容
- **"员工是新影响者"方向** —— 让工艺师 / 门店团队 / 后台员工直接成为品牌主叙事者
