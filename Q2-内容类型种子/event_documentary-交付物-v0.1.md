## 本文件是什么（核心定义）

- **交付物 ≠ 字段 schema**。交付物是一份"**玩法库 + 元素库 + 红线**"，是 AI 生产这种内容时可调用的**表达资源包**。
- **模板 ≠ 文字填空模板**。每张玩法卡里的「30 秒分镜参考」就是模板——**拍摄者照着能拍出来的执行卡片**。
- **AI 生产时不是"填字段"**，而是"**从玩法库里选一张卡片并展开**"。

> 定义来自用户 2026-04-13 裁决。详见 [_index.md](_index.md) §0。

## 本文件的"1-3 人小团队"放宽条款

**event_documentary 是企业叙事类 6 份里**首次使用"1-3 人小团队"放宽条款的文件**（process_trace 是第 2 个）。放宽原因：

1. **事件本质是协作的** —— 活动现场天然需要同时捕捉多个角度（前台/后台/观众/产品），单人无法完成
2. **事件跨时间** —— 从前夜准备到开场到散场，单人无法连续 24 小时跟拍
3. **事件有不可预期性** —— 突发瞬间需要有人替补机位

**但仍严守的边界：**
- ❌ 不允许专业摄制组 / 航拍 / 多机位导播
- ❌ 不允许专业灯光 / 专业收音麦克风阵列
- ✅ 最多 1-3 人同伴 + 手机/微单 + 现场自然光 + 最多 1 个领夹麦
- ✅ 后期仅用剪映 / CapCut 等手机端工具

## A. 北极星

**观众看完不只是"知道这场活动办了"，而是"我错过了现场但我感受到现场"——像真的到过一样感到事件的临场感、混乱感、紧张感、释然感，从而对品牌产生"它是活着的"的感受。**

> **三份原话备查：**
> - GPT5.4：不只是"这场活动办了"，而是感到事件的临场感、混乱感、紧张感、释然感——对品牌产生"它是活着的"的感受
> - 深度研究：我错过了现场但我感受到现场，同时对品牌产生"有活人、有能量、可信"的直觉
> - compass：从"打卡签到"到"72 小时时间线"——**事件的情绪曲线 > 事件的流程展示**

## B. 元素库（原子层）

### B1 视觉元素
- 倒计时镜头（开场前 60 / 30 / 10 分钟）
- 空场到满场的过渡
- 门槛镜头（门被推开瞬间）
- 散场后的"热闹尸体"（地上纸杯 / 歪的衣架 / 半开的试衣间门）
- 后台 vs 前台反差（同一时刻两个空间）
- 72 小时时间线跳切
- 定点延时（一个固定机位拍布展到撤展全程）
- 物件见证者视角（邀请函 / 签到笔 / 贴地胶带 / 收银台二维码）

### B2 听觉元素
- 空场的寂静 / 门外脚步声由远到近
- 门被推开的第一秒人声涌入
- 对讲机沙沙 / 催促 / "这件还有小码吗"
- 高跟鞋 / 衣架轻撞 / 试衣间门帘
- 散场的叹气 / 疲惫的笑声
- 单一声音主角（门铃 / 剪刀 / 扫码枪 / 鞋跟）
- 突然静音的 2 秒空镜

### B3 人物元素
- 后台手忙脚乱（店员深呼吸 / 捡别针 / 最后一次熨衣服）
- 前台只拍半个结果（一声掌声 / 一个笑着走出试衣间的顾客 / 一角闪过的新品）
- 人物群像（老板 / 店员 / 顾客 / 搭配师 / 搬货同事各 6 秒）
- 1-3 人小团队（允许）
- POV 第一人称（顾客视角 / 参与者视角）

### B4 场景元素
- 门店（开业 / 上新 / 快闪）
- 活动现场（发布会 / 展会 / 联名）
- 后台（试衣间 / 熨烫台 / 货架后 / 化妆间）
- 入口 / 门槛 / 转角
- 散场后的空间残留

### B5 情绪元素
- 前夜的紧张 / 逼近感
- 开场那一秒的吸气
- 中场的混乱密度
- 散场的后劲 / 空间温度感
- 反"圆满落幕"式平静收尾
- 失败视角（无人问津 / 首日零订单 / 布展翻车）

### B6 文字元素
- 时间戳（"开场前 60 分钟"）
- 一句式人物字幕（姓名/岗位 + 一句话）
- 倒叙字幕（"倒回 10 分钟前"）
- 结尾反鸡汤句（"灯关了，今天才算真正被记住"）
- 避免"圆满落幕 / 感谢到场"

### B7 时间元素
- 三段时间结构（前夜 / 此刻 / 余波）
- 倒叙开门（先给结果，再回到开门前）
- 72 小时时间压缩
- 定点延时一镜到底
- 非线性（散场开始 → 闪回）
- 反向时间线（高光 → 数周前筹备困难）

### B8 反差元素
- 前台体面 vs 后台真实
- 光鲜场面 vs 失败视角
- 热闹 vs 散场
- 准备的静 vs 开场的闹
- 预期（"燃炸音乐"）vs 现场（真实呼吸）

### B9 隐喻元素
- 门 = 事件的仪式开关
- 物件 = 沉默见证者
- 散场残留 = 事件的体温
- 声音 = 现场质地
- 后台 = 活动被一群活人硬托起来的证据

### C2 进阶玩法（少数高手在用）

#### 玩法卡 #C2-1 —— "倒计时纪实 + 前夜此刻余波三段结构"

| 字段 | 内容 |
|---|---|
| 档位 | ⭐ 进阶 |
| 一句话定义 | 不从"活动正式开始"拍起，而是从**开场前 60 / 30 / 10 分钟**开始——什么都还没发生，但一切都正在逼近。或用"前夜 / 此刻 / 余波"三个时间层组织：前夜打烊后挂最后一套衣服、此刻开门瞬间的人群呼吸声、余波活动后地上的胶带 |
| 北极星映射 | ✅ 事件最有戏的往往不是它顺利进行的时候，而是**它还没来得及发生、但所有人都在为它冒汗的时候**——让观众先感到"要开始了"，再去看"开始了" |
| 表达元素清单 | B1 时间倒数 / B2 后台杂音 / B3 手忙脚乱 / B1 空场到满场的过渡 / B2 碎片对话 / B1 门被推开瞬间 |

| 服装行业钩子 | ⭐⭐ 服装在这里是"待出场的演员"——被挂好、被熨平、被最后一次整理 |
| 实拍可行性 | 🟡 允许 1-3 人小团队 / 手机 + 稳定器即可 / 预算 0-100 元 / 单条 4 小时内可完成（但需要提前半天进场） |
| AI 辅助度 | ✅ 能做：倒计时时间节点建议 / 镜头清单 ; ❌ 不能做：替代现场紧张感 |

## D. 服装行业垂直机会

服装行业的 `event_documentary` 不该只拍"我们办了个活动"，而该拍**衣服第一次被集体目光照亮的瞬间**。服装品牌的活动不只是社交事件，也是"**产品从静物变成人和空间关系**"的瞬间。所以最该挖的不是舞台，而是**边缘**：试衣间门帘、临时熨台、抢尺码、顾客手摸面料的犹豫、店员最后一次把肩线拽正。

### 三个未被开发的角度

1. **"开场前 60 秒的空气"（C3-1 沉默叙事）** —— 事件最有戏的时刻是开场前，不是开场后
2. **"单声道事件"（C3-4 单声道纪实）** —— 用一种声音走完整场，比全景堆叠更有沉浸感
3. **"失败视角事件"（C3-7）** —— 服装行业是少数"敢展示失败就能赢信任"的行业之一

### 服装行业事件纪实方向（v0.2 M4 无痕处理：删去全部具体品牌 / 栏目 / 账号 / 数据 / 仅保留方向描述）
- **时装周"幕后故事"方向** —— 幕后情绪曲线代替前台热闹
- **直播间搬到溯源现场方向** —— 把溯源 / 牧场 / 棉田 / 工厂作为直播场景 / 消费者"云养殖 / 云访厂"
- **超级品牌日方向** —— 场景演绎 + 趋势解读 + 走秀三合一的综合叙事
- **沉浸式秀场方向** —— 薰衣草田 / 盐场 / 森林等非传统场所 / 手机拍摄的观众视角自发传播
- **名人 / KOL × 品牌跨界走秀方向** —— 借助第三方的"表演张力"把品牌放进流量场
- **第一人称家庭视角方向** —— 用家人 / 旁观者的第一视角讲品牌大事件
- **反向透明方向** —— "我们不怕展示品牌的傻气一面"

## F. AI 辅助边界

| 环节 | AI 能做 | AI 不能做 |
|---|---|---|
| 选题 | 活动前镜头清单（必须抓的 10 个瞬间）/ 时间节点建议 | 替代"这场活动有什么戏" |
| 脚本 | 倒计时结构 / 三段时间 / 反向时间线设计 / 字幕精简 | 替代故事线选择 |
| 拍摄 | 物件见证者候选 / 声音主角清单 / 人物群像角色表 | 替代现场临场判断 |
| 剪辑 | 自动打点粗剪 / 降噪 / 节奏建议 | 决定"哪个瞬间最值得保留" |
| 发布 | 标题 / 标签 / 封面 | 替代 FOMO 感的真实性 |

> **F 核心判断（GPT5.4 + 深度研究）：** 现场临场判断和"**哪里有戏**"必须靠人——AI 适合做"拍摄清单 / 时间结构 / 自动粗剪"，但故事线选择必须人为判断。
