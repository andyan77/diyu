## 本文件是什么（核心定义，防止脱离索引被误读）

- **交付物 ≠ 字段 schema**。交付物是一份"**玩法库 + 元素库 + 红线**"，是 AI 生产这种内容时可调用的**表达资源包**。
- **模板 ≠ 文字填空模板**。每张玩法卡里的「30 秒分镜参考」就是模板——**拍摄者照着能拍出来的执行卡片**，含画面 + 声音 + 动作 + 节奏。
- **AI 生产时不是"填字段"**，而是"**从玩法库里选一张卡片并展开**"。

> 定义来自用户 2026-04-13 裁决。详见 [_index.md](_index.md) §0。

## ⚠️ 本文件的特殊结构：兵团向口径（观众切换）

### 1. 为什么 training_material 需要口径切换

前 16 份 ContentType 交付物（batch 1 的 7 份 + batch 2 的 6 份 + batch 3 前 3 份）的**观众都是消费者**—— 文案的目标是"让陌生人看完想买 / 让进店客人决定买 / 让老顾客持续复购"。

**`training_material` 的观众是内部人员**—— 员工 / 加盟商 / 店长 / 新员工。目标完全不同："**员工看完下次真遇到场景时，脑子里自动跳出正确动作**"。这决定了所有字段语义都要切换。

### 2. α 改良版字段语义扩展（兵团向版本）

继承 `product_copy_general` 首创的 α 改良版思想——**保留 10 字段结构不拆表**，但本文件内部对字段语义做针对性调整：

| 字段 | 消费者向（前 16 份）| **兵团向（本文件）**|
|---|---|---|
| **北极星映射** | 观众看完后愿意买 / 愿意信 / 愿意收藏 | **员工看完后知道下次怎么做 / 记得住 / 愿意照做**|
| **服装行业钩子** | 服装作为情绪容器 / 身体界面 / 审美符号 | **服装作为可被训练的操作对象**（折叠 / 挂装 / 蒸烫 / 量体 / 陈列 / 量体的可视化动作）|
| **实拍可行性** | 1 人 1 手机拍生活场景 | 1 人 1 手机拍**门店真实操作台 / 试衣间 / 后仓**（要求同 + 场地是企业自有）|
| **北极星判定** | 观众的购买意向 / 情感共鸣 | **员工的行为改变 / 操作正确率 / 记忆保持**|
| **AI 辅助度** | 结构 + 文案 + 分镜 | **题库生成 + 情境脚本 + 错题点汇总**——但"正确动作标准"必须由业务最懂的人把关 |

### 3. 和前 16 份的硬边界

| ContentType | 本 CT 的边界 |
|---|---|
| `knowledge_sharing`（batch 1）| batch 1 的 knowledge_sharing 是"**教消费者穿搭 / 面料 / 选购**" ; 本 CT 是"**教员工怎么接待消费者 / 怎么操作商品**" —— 同样是"教"，但**教的人不同 / 学的人不同** |
| `role_work_vlog`（batch 2）| role_work_vlog 是"**把岗位工作展示给消费者看**"（设计师日常 / 样衣工日常）——还是消费者向 ; 本 CT 是"**把岗位工作教给新人**"——兵团向 |
| `store_daily`（batch 3）| store_daily 是"**让消费者看见门店生命力**" ; 本 CT 是"**让员工看见自己门店里的高发错误**"（C3-3 神秘顾客回放）|

### 4. 兵团向的 3 个核心原则（来自认知负荷理论 + 测试效应研究）

1. **看懂 ≠ 会做** —— 培训物料必须有**动作层面的示范**，不能只讲道理
2. **先做判断再给答案** —— 场景训练 / 先错一次再纠正 / 比看标准答案更接近上岗（**测试效应 testing effect** 是经典认知科学结论）
3. **微学习 + 即时资源** —— 单条 ≤ 90 秒 / 3-5 分钟可吸收窗口 / 移动端可达 —— 不要一次讲完所有流程

## A. 北极星

**员工 / 加盟商看完后，不是"觉得讲得不错"，而是"**下次真碰到场景时，脑子里会自动跳出正确动作**"—— 我不困 / 我知道下一班 / 下一单怎么做 / 我能记住关键点 / 我愿意照做，因为它省事 / 清楚 / 能让我少背锅。**

> **三份原话备查：**
> - 通用类.md：下次真碰到场景时 / 脑子里会自动跳出正确动作
> - 通用类compass.md：90% of training content is forgotten within 6 months —— 培训的真实问题不是"员工不认真"，而是"内容本身不可调用"
> - 通用类deep-research-report.md：我不困 / 我知道下一班 / 下一单怎么做 / 我能记住关键点 / 我愿意照做 因为它省事 / 清楚 / 能让我少背锅

## B. 元素库（原子层）

> **元素库的兵团向特殊性**：本 CT 的元素服务于"**教员工做对一个动作 / 记住一个判断**"，不是"让消费者产生购买意向"。所有元素都是**可被拍成动作示范的**素材，不是审美素材。

### B1 视觉元素（动作示范层）
- **俯拍操作台**（折叠 / 蒸烫 / 打包 / 贴吊牌 / 装袋 —— **俯拍是兵团动作教学的黄金机位**）
- 手部特写（动作关键点 / 错误姿势 / 正确姿势对比）
- 第一视角（"你就是店长 / 导购"的代入镜头）
- 试衣间外情境（脚部 / 背影 / 帘子 —— 不拍顾客脸的隐私保护）
- 红框 / 箭头 / 圆圈标注（错题本剧场核心）
- 画面定格 + 字幕问题（"你看出错在哪了吗?"）
- 前后对比构图（错误 5 秒 + 正确 10 秒）
- 步骤编号字幕（"第 1 步 / 第 2 步 / 第 3 步"）
- **避免**：企业宣传片式全景扫店 / 模特大片 / 审美化拍摄

### B2 听觉元素
- 动作声（折叠声 / 蒸汽声 / 衣架碰撞声 / 胶带声）
- 短提示音（测验倒计时 / 答案揭示）
- 极短旁白（"记住这里 / 注意这一步"）
- **无 BGM 或极弱环境音**（培训不是娱乐）
- **避开**：长旁白讲解（超过 15 秒的口播）/ 热门 BGM / 企业宣传片式配乐

### B3 人物元素
- 店长 / 资深员工出镜（**真实业务口吻**）
- 只露手的操作示范（最常用）
- 新员工作为"学员"角色（第一人称代入）
- 同事扮演顾客（不使用真实顾客 / 避免隐私问题）
- 连载角色（新来的小陈 / 老店员阿琳 / 挑剔型顾客 / 犹豫型顾客）
- **避免**：专业演员 / 模特 / 营销口吻

### B4 场景元素
- **门店操作台**（折叠区 / 打包区 / 熨烫区）
- 试衣间外（真实情境训练场）
- 货架前 / 镜前 / 收银台
- 后仓 / 盘点区
- 班前会场地（店长 30 秒微教练的场景）
- **关键**：场景必须是**真实的工作场景**，不是布置的摄影棚

### B5 情绪元素
- 克制感（培训不煽情 / 不戏剧化）
- 判断感（"你会怎么选?"的思考停顿）
- 错误后的具体反馈感（"原来是这样错的"）
- 可复用感（"这条我下次可以照做"）
- **避开**：戏剧化表演 / 企业激励型煽情 / "加油干"式口号

### B6 文字元素
- 题目字幕（"顾客试穿后说'有点扎'，你第一句回什么?"）
- 选项字幕（"A: / B: / C:"）
- 倒计时字幕（"3 / 2 / 1"）
- 关键动作短指令（"对齐 / 放平 / 封口"）
- 错误标注（"这里错了 / 为什么错"）
- 步骤编号（"第 1 步 / 第 2 步"）
- **避免**：长段文字 SOP / 制度条款 / 抽象术语

### B7 时间元素
- **极短时长**（单条 ≤ 90 秒 / 目标 3-5 分钟学习吸收窗口）
- 班前 5 分钟（店长微教练时段）
- 新员工前 7 天（入职连载剧场）
- 每日班前刷 3 条（微学习串联）
- 错题练习 + 反馈的 30 秒循环（测验 10 秒 + 答题 10 秒 + 反馈 10 秒）

### B8 反差元素
- 错误示范 vs 正确示范（同一动作 / 同一人 / 同一机位）
- 顾客说的话 vs 顾客真正想表达的
- 员工 A 的反应 vs 员工 B 的反应（同情境不同应对）
- 第一反应 vs 正确判断（"先别急着答 / 先想 3 秒"）
- 理论话术 vs 现场发挥（"话术卡说 XX / 但实际要根据顾客语气"）

### B9 隐喻 / 认知元素
- 培训 = 工作流中的即时提醒（不是课堂）
- 员工记忆 = 动作肌肉记忆（不是 U 盘存储）
- 培训内容 = 可调用动作库（不是制度手册）
- 错误 = 危险点雷达（不是惩罚）
- 微学习 = 班前刷三条（不是 20 分钟讲课）

### C2 进阶玩法（少数高手在用）

> **本档延续 product_review 首创的 β 标签方案**（每张卡标「适用场景」）**+ 本 CT 首创「学习机制」字段** —— 每张进阶 / 突破性玩法卡必须标注其背后的认知科学机制（testing effect / worked example / microlearning / scenario-based learning / retrieval practice），让 AI 生成时能按学习目标匹配正确机制，而不是按"好看程度"选卡。

#### 玩法卡 #C2-1 —— "分支情境训练 / 先判断再给答案"

| 字段 | 内容 |
|---|---|
| 档位 | ⭐ 进阶 |
| **适用场景** | **接客为主**（分支情境直接对标"店内遇到真实顾客"；引流层不涉及）|
| **学习机制** | **Scenario-based learning + Testing effect**——Elucidat / FLOWSPARKS / Axonify 都把 branching scenario / "choose best response" / 即时反馈作为核心微学习格式 ; 认知科学上"先测试后讲解"比"先讲解后测试"记忆保持更好 |
| 一句话定义 | 给员工一个**真实场景**："顾客说太贵 / 顾客想退换 / 顾客进门只看看不说话"，然后给 2-3 个应对选项，**选完立刻反馈**。先让员工做判断 / 再告诉他为什么 |
| 北极星映射 | ✅ 员工学到的不是"标准答案"而是"**判断顺序**" —— 遇到新场景时能迁移 |

## D. 服装行业垂直机会

服装行业培训物料**长期被两种思路绑架**：一种是**总部下发的"品牌词典 + 卖点手册"**（C1-2 话术灌输 / C1-3 长视频一次讲完），一种是**散落在群里的临时通知和长图制度**（compass §4 引用 / 一刀切 + 阅读成本极高）。**真正的机会在于把培训内容做成"可调用动作"** —— 不是文字 SOP / 不是 PPT 录屏 / 而是**让员工的"手"和"判断顺序"被持续训练**。

### 三个未被开发的角度

1. **"动作可视化 + 微学习窗口"的 SOP 训练**（C2-2 Do/Don't / C2-4 Worked example / C3-4 SOP ASMR / C3-6 错题本剧场）—— 服装行业**大量技能是手部和空间动作**（折叠 / 挂装 / 蒸烫 / 量体 / 陈列 / 打包 / 吊牌检查），俯拍特写是**兵团动作教学的黄金机位** / 比任何文字 SOP 高 10 倍效率
2. **"先错再对 + 情境决策"的判断力训练**（C2-1 分支情境 / C3-1 错误先发生 / C3-5 门店 RPG / C3-7 翻车 stand-up）—— 服装销售的胜负**几乎全在 3 秒决策点**（顾客皱眉 / 沉默 / 反复照镜子 等非语言信号），抽象规范无法训练这种"**听言外之意**"能力 / 只有情境演练 + 后果可视化能突破
3. **"自下而上员工 UGC + 同伴互评"的知识沉淀**（C3-8 你来当店长 / C3-9 班前会 30 秒）—— 服装零售**一线智慧密度极高**但从未被系统提取，UGC 微课和同伴评议是**把隐性知识变成组织资产的最低成本路径** / 同时建立兵团文化和职业认同感

### 行业趋势支撑

- **微学习窗口压缩**（compass §4）：传统 20-60 分钟模块 → **当前最佳模块长度 < 7 分钟** / 数据显示**90% 的培训内容在 6 个月内被遗忘** —— 内容设计必须从"完整讲解"切换到"即时调用"
- **测试效应 testing effect**（deep-research §training_material 引用）：**先测试再讲解**比"先讲解后测试"记忆保持显著更好 / 课堂场景中低风险小测验也能提升章节与学期考试表现
- **认知负荷理论 cognitive load theory**（deep-research §training_material 引用）：**工作记忆容量有限** / 信息过载会导致学习崩溃 / 教学设计必须**减少冗余 + 避免一次塞太多**
- **微学习系统综述**（deep-research §training_material 引用）：在所综述研究中**对认知 / 行为 / 情感层面均呈现正向影响** / microlearning 不再是猜测而是被验证的方法
- **众包微课开发**（compass §4 邱昭良引用）：**人人参与开发与共享**内容 / 快速提取经验性知识 —— 是培训理论的范式转移
