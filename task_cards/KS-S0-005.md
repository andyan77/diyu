# KS-S0-005 · ContentType canonical id + alias map

- **工程**：建立 `content_type_canonical.csv`：`canonical_content_type_id, name_zh, aliases, coverage_status`。
- **S gate**：S0；为 §3.2 content_type_view、KS-RETRIEVAL-002 路由提供基础。
