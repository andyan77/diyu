一、正式定位

本包正式作为冷启动知识素材包使用，主落点冻结为 FabricKnowledge（面料知识） 与 CraftKnowledge（工艺品控）；派生回填对象冻结为 StylingRule（搭配规则）、Product（商品）、TrainingMaterial（培训材料）。仓库真源已经把 L2 服装专业知识中的“面料知识”“工艺品控”明确列入正式建设范围，同时 MerchandisingSkill（陈列搭配技能） 已明确消费 Product（商品）、StylingRule（搭配规则）、TrainingMaterial（培训材料） 这三类对象。

本版的“正式”，指的是知识素材包口径正式冻结；不等同于 FabricKnowledge（面料知识）、CraftKnowledge（工艺品控） 的字段 schema 已在仓库里最终冻结。字段层面后续仍应按 KnowledgeExtractionRecord（候选知识包） → 映射入正式对象的管线落库，且继续遵守“证据优先、禁止推断”的抽取纪律。

本包的人设承接口径同步冻结为 product_professional（商品专业者），并由 designer（设计师）、sample_worker（样衣工）、merchandising_director（商品总监）、design_director（设计总监） 等 RoleProfile（岗位画像） 提供话语来源；也就是这包不是“百科知识”，而是后续要能被专业人设讲出来、被培训物料教出去、被商品测评用得上的判断包。

二、本包正式回答什么

本包正式回答五件事：

第一，哪些面料差异会直接改变推荐结论。
第二，哪些工艺细节会改变陈列或卖点表达。
第三，哪些材质最容易被顾客误解。
第四，哪些推荐看似合理、实际穿后翻车。
第五，哪些品质风险必须提前说清。

这包的作用不是把 Q7-Q12 再抽象讨论一遍，而是把“审美判断”往“真实穿着、真实成交、真实售后、真实培训”这四个方向压实。仓库真源已经把 training_material（培训物料） 的目标写得很清楚：不是让人“觉得讲得不错”，而是让员工“下次真碰到场景时，脑子里自动跳出正确动作”。所以本包的所有结论都按“可训练、可接客、可解释、可避坑”来写。

三、五个必答问题的正式结论
1）哪些面料差异会直接改变推荐结论

真正会改推荐结论的，不是“天然纤维 / 化纤”这种大词，而是六个判断：贴不贴肉、垂不垂、支不支、透不透、热不热、洗后稳不稳。仓库真源对 FabricKnowledge（面料知识） 也已经把重点收敛到手感、透气 / 吸湿 / 保暖、适用季节、适合版型、洗护与常见问题。

具体到门店推荐，最会改结论的是这几种冲突：

软垂 vs 挺括，会直接改“显肉还是修形”；
轻透 vs 遮蔽，会直接改“能不能单穿”；
高吸湿 / 凉感 vs 易皱 / 易挂长，会直接改“适不适合通勤、差旅、久坐”；
表面细腻 vs 摩擦易老，会直接改“适不适合背包党、带娃党、久坐党”；
洗护门槛低 vs 高，会直接改“能不能放心卖给回家直接扔洗衣机的顾客”。TENCEL™ Lyocell（天丝莱赛尔） 官方也把高吸湿和更高体感舒适度列为核心特征之一。

2）哪些工艺细节会改变陈列或卖点表达

门襟 / 领口 / 驳头 / 裤头有没有衬，缝口平不平，明线直不直，褶子稳不稳，条格对不对，洗后会不会歪、会不会缩，这些都会直接改变陈列面相，也决定“挺、垂、利落、讲究、耐穿”这些词能不能说出口。棉类 shrinkage（缩水）和 skewing（扭斜）不只是尺寸问题，还会连带影响缝位、辅料关系和上身 fit（合体感）。

3）哪些材质最容易被顾客误解

最容易被误解的不是“稀有材质”，反而是店里最常见、名字又最容易被混着说的材质：
真丝缎 vs 聚酯缎；
天丝 / 莫代尔 / 粘胶 vs 被统称成“冰丝”；
羊毛 / 羊绒 vs 只看名字不看比例；
纯棉 vs 被误认为一定不缩水、不透、不变形；
亚麻 vs 被误认为一定扎、或者一定高级但完全不皱；
功能化纤 vs 被一句“化纤”直接判死刑。
这类误解会直接把门店解释带偏，也会把售后矛盾提前埋进去。knowledge_sharing（知识分享） 与 product_review（产品测评） 真源里也已经把“面料 / 工艺 / 洗护 / 现场证据”列成可直接讲的高价值知识方向。

4）哪些推荐看似合理，实际穿后翻车

最典型的翻车，不是“完全搭错”，而是镜前一站好看，穿一天不成立。
比如把缎面直身裙推给对腹部、内裤痕、光泽反射很敏感的人；
把亚麻衬衫推给全天开会、久坐、还要求下班也平整的人；
把高垂粘胶裙推给需要轮廓清楚、动作多、在意贴肉的人；
把深色原牛仔推给怕勒、怕热、怕掉色、怕回缩的人；
把未说明洗护门槛的羊毛针织推给“一律机洗烘干”的顾客。
这些都不是审美错，是穿着情境判断错。

5）哪些品质风险必须提前说清

必须前置说清的风险，正式冻结为八类：缩水、起球、褪色 / 沾色、透感、勾丝、静电、摩擦位起毛 / 发亮、机洗 / 烘干限制。尤其 wool（羊毛）类能不能机洗，不能靠销售经验拍脑袋，必须先看 care claim（洗护标 / 洗护声明）；Woolmark（国际羊毛局）也明确写明：只有当洗护标注明可机洗时，才可按相应程序机洗。

四、首批 FabricKnowledge（面料知识） 正式冻结清单
1. 棉针织 / 棉弹类

这类的核心价值是亲肤、吸汗、容易被接受，但最大误区也是“因为熟，所以被低估”。
体感上，薄款偏贴身，厚款偏稳，垂感通常一般，更多是贴着身体走，不靠它做悬垂感。适合 T 恤、打底、家常通勤、轻休闲。
最容易误导顾客的点有三条：纯棉不等于不透；纯棉不等于不变形；纯棉也不等于穿久了还保持领口利落。
正式风险提示冻结为：领口松、下摆卷、白浅色透、汗后贴身、机洗后回缩、针织类更容易出现尺寸波动。CottonWorks（棉花公司 CottonWorks）也明确指出 shrinkage（缩水）会影响 fit（合体感）、seam puckering（缝口起皱）、skew（扭斜）以及面辅料关系。

2. 亚麻 / 棉麻类

这类真正卖的不是“精致”，而是“干爽、空气感、松弛感”。
体感偏干、透气，表面颗粒感明显，支撑感通常比顾客想象中更强，垂感不算软塌。适合夏季通勤、度假、轻文艺、宽松轮廓陈列。
最容易误导的是把它讲成“高级且全天不皱”。正式口径应改成：它的高级感，很大一部分恰恰来自自然折痕和纤维表情。
正式风险提示冻结为：肘部、臀部、坐位、前门襟容易出褶；版型越直、越薄、越浅色，越要把透感和皱感提前说清。

3. 粘胶 / 莫代尔 / 天丝类

这类最容易在镜前赢，但也最容易在穿后暴露问题。
体感通常软、凉、顺，垂感好，容易把裙装、衬衫、阔腿裤做出“会流动”的感觉。适合需要柔和、轻盈、显垂的场景。
最容易误导的是把“软垂”说成“修形”。很多时候它只是顺着身体线条走，不会帮顾客藏掉她在意的位置。
正式风险提示冻结为：贴肉、挂长、缝位变形、薄款暴露体表痕迹、浅色透、久穿后局部发亮。Lenzing（兰精）对 TENCEL™ Lyocell（天丝莱赛尔） 的官方描述也把高 moisture uptake（吸湿）和更高 thermal comfort（体感舒适度）列为核心特性。

4. 真丝 / 醋酸 / 缎面类

这类不是“显贵万能牌”，而是“对穿着状态很诚实”的面料。
体感通常滑、凉、轻，垂感漂亮，但对腹部、内衣边、肩带、灯光反射都很敏感。适合晚宴、轻礼服、精致通勤、拍摄表达。
最容易误导的是只卖光泽，不讲暴露感。
正式风险提示冻结为：勾丝、水痕、压痕、静电、浅色透、局部发白、体表痕迹明显；这类面料推荐时必须同步判断顾客对“痕迹暴露”的容忍度。

5. 羊毛 / 羊绒 / 毛混类

这类卖点不是“软”两个字，而是保暖、弹性回复、表面质感和体面感。
体感从细软到略扎都有，垂感和支撑感取决于纱支、织法和克重。适合秋冬通勤、大衣、针织、外套。
最容易误导的是“含毛=高级”“羊绒=永远不起球”“毛类=都要手洗”。
正式风险提示冻结为：摩擦位起球、包带处先老、局部起毛、毡缩、刺痒、肩肘膝鼓包。能否机洗必须看洗护标；Woolmark（国际羊毛局）明确说明，只有洗护声明允许机洗的 wool garment（羊毛服装）才可安全机洗。

6. 牛仔 / 斜纹重棉类

这类最典型的特点是镜前立得住、陈列也有型，但真正的成交判断不该只看镜前。
体感偏实、偏硬，支撑感强，轮廓清晰；适合裤装、工装、短外套、休闲通勤。
最容易误导的是“显瘦有精神”就直接推，忽略久坐、蹲起、闷热、掉色和回缩。
正式风险提示冻结为：腰腹压迫、膝盖鼓包、裆部紧、深色掉色、未经充分预洗时存在回缩和扭斜风险。

7. 雪纺 / 欧根纱 / 轻透材

这类更适合做层次工具，不适合默认按“主面料安全款”去卖。
雪纺偏软垂，欧根纱偏支撑、带空气感；共同问题是透、轻、勾、静电。适合叠穿、礼服、罩层、氛围陈列。
最容易误导的是把它当成“显瘦遮肉”单穿主力。
正式风险提示冻结为：透感、静电、勾丝、挂纱、缝口起皱；精细轻薄材质本身就更容易出现 seam puckering（缝口起皱）问题。

8. 涤纶 / 锦纶 / 功能化纤类

这类不能用“化纤”一刀切。它真正的分水岭在纱线、织法、后整理和用途。
优点是稳定、轻、易打理、功能表达明显，适合旅行、防风外套、运动休闲、需要形态稳定的单品。
最容易误导的是只讲“好打理”，不讲闷、静电、摩擦发亮、面料噪音。
正式风险提示冻结为：静电、热感、发亮、摩擦位先老、表面起毛、里外层手感落差大。

五、首批 CraftKnowledge（工艺品控） 正式冻结清单
1. 预缩 / 水洗 / 定型

这组正式定义为“售后前置工艺”。
它直接决定一件衣服洗后会不会缩、歪、扭、变短，决定裤线、裙摆、袖长会不会洗一次就走样。
门店说这件“耐穿、稳定、好打理”之前，必须先判断预缩、水洗和定型是否成立。CottonWorks（棉花公司 CottonWorks）明确指出 shrinkage（缩水）受面料结构、湿处理、整理以及成衣制造过程共同影响。

2. 缝制质量：线迹、跳针、缝口起皱

这组正式定义为“近看质感工艺”。
顾客不一定会说 seam puckering（缝口起皱）这个词，但她一定能看出“这件怎么有点抽、歪、鼓、皱”。
Coats（高士）对 seam puckering（缝口起皱）的说明很清楚：它不只是线的问题，还和 fabric structure（面料结构）、seam construction（缝型）、thread / needle size（针线规格）以及 feeding problems（送布问题）有关。

3. 粘合衬 / 贴合 / 复合

这组正式定义为“轮廓成立工艺”。
领口、门襟、驳头、裤头、袖山，一旦衬用错或贴合不稳，陈列时还能撑，穿几次就可能翻边、起泡、脱层。
凡是要用“挺、立、利落、骨架感”做卖点的单品，都要先过这组判断。

4. 染整 / 色牢度

这组正式定义为“日常使用风险工艺”。
不是只有深色才要看色牢度，浅深拼接、浅色内搭、包带摩擦、汗液摩擦都要看。
凡是顾客会长期背包、贴身穿、或者浅深叠穿的单品，都必须把褪色 / 沾色风险前置说明。

5. 磨毛 / 拉毛 / 起绒 / 防起球

这组正式定义为“第一触感与老化速度工艺”。
很多单品上手很讨喜，但三次之后就显旧，问题往往不在设计，而在表层处理和摩擦位耐受。
凡是卖点写“软、糯、绒、暖、亲肤”的单品，都要同步判断背包位、腋下、袖口、内侧会不会先起毛、起球、发旧。

6. 压褶 / 明线 / 对条对格 / 细节收口

这组正式定义为“讲究感工艺”。
它最直接决定陈列是不是高级、近看是不是用心。百褶稳不稳、线迹直不直、格纹对不对、滚边净不净，顾客不一定会说术语，但一定会据此把衣服归类成“讲究”还是“将就”。
凡是要卖“做工感、品质感、工艺感”的单品，这组必须先看。

六、首批高频误判，正式冻结为培训与接客重点
误判 1：把“软”当成“高级”

正式纠偏：软只说明触感，不说明轮廓成立。软垂类面料要同步判断贴肉、挂长、暴露体表痕迹风险。
落盘对象：StylingRule（搭配规则）、TrainingMaterial（培训材料）。

误判 2：把“重”当成“有质感”

正式纠偏：重可能只是厚、闷、压身；质感要看表面、回弹、结构、缝制和使用后的稳定性。
落盘对象：Product（商品）、TrainingMaterial（培训材料）。

误判 3：把“会皱”当成“低级”

正式纠偏：有些皱是材质特性，不是低级；关键是顾客是否接受这种表情，以及场景是否允许。
落盘对象：FabricKnowledge（面料知识）、TrainingMaterial（培训材料）。

误判 4：把“有光泽”当成“显贵”

正式纠偏：光泽会同步放大体表痕迹、褶印、贴肉和廉价感；不是所有光泽都显贵。
落盘对象：StylingRule（搭配规则）、Product（商品）。

误判 5：把“纯棉”当成“零风险”

正式纠偏：纯棉也会透、会卷、会缩、会变形；针织棉更要看领口、下摆和洗后稳定性。
落盘对象：FabricKnowledge（面料知识）、TrainingMaterial（培训材料）。

误判 6：只讲成分，不讲穿着情境

正式纠偏：成分只是起点，真正决定成交的是场景、动作、出汗、久坐、洗护、摩擦。
落盘对象：Product（商品）、product_review（产品测评）。

七、必须提前说清的风险提示，正式冻结为门店统一口径

正式统一口径如下：

能不能机洗，看洗护标，不看感觉。
会不会缩水，看预缩 / 水洗 / 定型，不看“摸起来稳不稳”。
会不会起皱，看纤维、织法、整理，不看它挂着时平不平。
会不会透，看颜色、厚度、织法、光线，不看静态陈列。
会不会起球，看摩擦场景，不只看材质名。
会不会勾丝，看表面精细度和使用场景，不只看价格。
会不会显廉价，看面料表情、缝制、收口、光泽控制，不看营销词。
会不会翻车，看顾客穿着习惯和容忍度，不看镜前第一眼。
