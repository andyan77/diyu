一、正式定位

“岗位手感与一线判断包”不是新的独立实体，也不是空泛的方法论。
它的正式定位是：

作为首批 7 个 RoleProfile（岗位画像）的高价值知识内容包
作为 TrainingMaterial（培训材料）的带教底稿
作为 StylingRule（搭配规则）的来源证据层
用来解决“系统会写，但写得不像干过的人”这个冷启动硬伤。
二、正式范围

本包正式覆盖以下 7 个 RoleProfile（岗位画像）：

store_manager（门店店长）
warehouse_supervisor（仓库主管）
sales_associate（门店店员 / 导购）
designer（设计师）
sample_worker（样衣工）
design_director（设计总监）
merchandising_director（商品总监）

其中冷启动优先收口顺序正式采用：

P0：sales_associate（门店店员 / 导购）、store_manager（门店店长）、designer（设计师）、sample_worker（样衣工）
P1：warehouse_supervisor（仓库主管）、design_director（设计总监）、merchandising_director（商品总监）
三、正式采集与入库原则
1. 先收“这个岗位真的怎么干”，不先收“这个岗位应该怎么说”

正式优先级固定为：

真实动作
真实判断
真实困境
真实话术
真实情绪
最后才是总结表达。
2. 每条岗位知识必须能回填正式对象字段

一条素材至少要能稳定支撑以下字段中的两类以上：

authenticity_anchors（真实性锚点）
topic_scope（话题边界）
typical_scenarios（典型场景）
emotional_spectrum（情绪光谱）
role_type（岗位类型）/ 岗位动作层描述。
3. 素材必须带证据形态

统一只认 4 类证据：

oral_evidence（口述证据）：访谈、复盘、对话记录
action_evidence（动作证据）：视频、照片、现场步骤
document_evidence（文档证据）：SOP、工艺单、培训资料、表格
physical_evidence（物证）：样衣、吊牌、货架、退货单、调货记录。
4. 失败素材必须主动入库

岗位知识如果只有“做得好”，最后一定长 AI 味。
正式要求必须主动采：

做砸过什么
最怕什么
最常卡在哪
哪句话最不该说
哪种判断最容易失手。
四、正式内容结构

本包正式由 5 个子包组成，每岗统一执行：

A. 岗位日常动作包

回答这个岗位平时到底在做什么，不是职责表，而是动作链。

B. 岗位判断句包

回答这个岗位凭什么判断、看什么细节下结论。

C. 高频问题与失败案例包

回答这个岗位最常失手在哪里，哪些坑新人最容易踩。

D. 典型场景与时间节点包

回答这个岗位在哪些时段、场景下最能显出手感。

E. 特有物件 / 空间 / 证物包

回答这个岗位的真实感从哪些物件、空间、痕迹里长出来。

五、7 个岗位的正式收口
1. store_manager（门店店长）
岗位手感

老店长不是先看“这件衣服漂不漂亮”，而是先看今天这家店的节奏顺不顺。
先看货场有没有主次，再看店员有没有站位，再看顾客有没有停顿，最后看后场和库存能不能接住。这个岗位的核心不是会卖，而是让整家店不乱。
对应素材真源已收口为：开店前检查、闭店后复盘、顾客高峰期排班 / 顶岗、新人带教、陈列调整、补货、临时调货、处理投诉与犹豫客。

一线判断

老店长一眼先看四件事：

顾客今天是来逛，还是来确认
高峰期该先稳团队，还是先冲销量
一家店“今天状态不对”最早从哪里冒出来
哪些话术 / 动作会让前台直接失手。
不能教条化的经验
不是每个犹豫客都该加压成交
不是每个活动日都该把爆款推到最前
不是陈列好看就等于卖得动
不是新人热情就等于接客有效
首批硬证据
一次真实闭店复盘记录
一次高峰期救火经过
一份“新人最容易犯错点”清单。
正式落点

优先进入 RoleProfile（岗位画像），并直接派生给 TrainingMaterial（培训材料）的“店长带教 / 复盘 / 班前会”模块。

2. warehouse_supervisor（仓库主管）
岗位手感

仓库主管的手感不在“会整理”，而在能不能让前场承诺和后场事实对得上。
他看的不是一件货好不好看，而是货态、尺码、断码结构、补货节奏、账实一致、调拨时点。
对应素材真源已收口为：入库 / 出库、调货、找货 / 补货 / 返仓、断码处理、爆款补货优先级判断。

一线判断

这个岗位最核心的判断是：

什么货一看就知道容易压
哪种断码最伤门店节奏
什么时候优先保速度，什么时候优先保准确
少一件货的时候，问题是粗心还是流程没立住。
仓库最怕前场怎么乱推
不看库存深度就承诺成套
断码当齐码卖
调货周期没确认就答应顾客
替代款按同款卖
前台卖法和后场实际货况脱节
不能教条化的经验
不是库里“还有”就等于前台能卖
不是爆款都值得优先补深
不是速度永远压倒准确
不是临时救火就能替代稳定流程
首批硬证据
一次真实调货失误复盘
一份“仓库最怕出现的 5 种情况”清单
一段后仓找货 / 补货真实动作视频。
正式落点

优先进入 RoleProfile（岗位画像），并派生给库存类 TrainingMaterial（培训材料）；涉及库存边界的部分，可为库存 / 替代相关规则提供证据，但暂不直接写死为 StylingRule（搭配规则）。

3. sales_associate（门店店员 / 导购）
岗位手感

导购真正的手感，不是嘴甜，而是会读顾客的停顿和身体反馈。
她要判断顾客是在试探、在拒绝、在防御，还是在等一个更贴身的解释。
这个岗位同时承接前台接客与私域出口位，是 styling_advisor（穿搭顾问）的人设底座，也是朋友圈 / 微信群 / 私信这类 PrivateOutletChannel（私域出口位）的主输出源。

一线判断

导购最关键的判断包括：

顾客是在试探你，不是在拒绝你
一套搭配为什么成立
什么时候不要再推
什么顾客先给场景，什么顾客先给身体感受
私域里哪种内容会让老客反感，哪种内容会让她回复。
新人最容易学歪
把“专业”学成背参数
把“热情”学成逼单
把“搭配建议”学成审美输出
把朋友圈经营学成广告轰炸
把试穿建议学成统一台词
哪些经验只能参考，不能写硬规则
顾客说“再看看”就一定没戏
顾客先问价格就是预算低
圈里多发货图就一定有转化
试穿满意但不成交一定是嫌贵
首批硬证据
一次真实试穿建议全过程
一份“不能这样发朋友圈”反面清单
一次老客被重新唤醒的私域触达案例。
正式落点

这是本包里最直接进入 TrainingMaterial（培训材料）和 StylingRule（搭配规则）证据层的岗位。
其中“看身形 / 看场景 / 看体感 / 看停顿”的部分，可转成搭配与试穿建议规则；“私域怎么发不惹人烦”的部分，只进培训，不写成硬规则。

4. designer（设计师）
岗位手感

设计师的手感不是“会想”，而是知道一件衣服的问题到底出在版型、面料还是细节。
她要能判断：什么看起来好看但上身不成立，什么灵感适合品牌，什么必须砍掉。
真源已收口的素材包括：灵感记录、面料挑选、版型初判、样衣评估、修改建议。

一线判断
一件衣服的问题出在版型、面料还是细节
为什么要砍掉自己喜欢的方案
什么灵感属于品牌，什么不属于
设计逻辑什么时候会和商品逻辑冲突
新人最容易学歪
把“审美好”当成“设计成立”
把“灵感”当成“可落地”
把“改版多次”理解成不专业
把“坚持自我”理解成不听反馈
首批硬证据
一次样衣不成立的完整复盘
一组 1.0 / 2.0 / 3.0 版本对照
一段“为什么改掉这一版”的原话。
正式落点

主要进入 RoleProfile（岗位画像）和 product_professional（商品专业者）侧的内容底座；其中版型 / 面料 / 细节判断，可为专业型 StylingRule（搭配规则）提供来源证据。

5. sample_worker（样衣工）
岗位手感

样衣工的手感最能把“专业”从嘴上拉回手上。
这个岗位看的是：哪种设计一看就知道难做，哪一步最容易翻车，哪种缝法会让成衣质感直接掉下去。
真源已收口的素材包括：打样流程、裁剪、缝制、返工、设计要求如何转成手上动作。

一线判断
做出来和做得好是两回事
返工不是丢脸，是工艺判断的一部分
设计图能不能落地，要看手上动作能不能成立
线迹、裁片、返工痕迹都是证据，不是边角料
新人最容易学歪
只追求完成，不追求质感
只看图，不看工艺单
返工时只补表面，不补根因
以为快就是熟练
首批硬证据
一次返工全过程
一组“正确线迹 / 失败线迹”对照
一段“这一步最烦、最容易出错”的原话。
正式落点

主要进入 RoleProfile（岗位画像）和工艺向 TrainingMaterial（培训材料）；与工艺、版型、面料直接相关的部分，可作为 StylingRule（搭配规则）和专业知识分享的证据来源。

6. design_director（设计总监）
岗位手感

设计总监不是“更资深的设计师”，而是品牌审美边界和方案取舍的裁决人。
这个岗位最值钱的不是创意，而是判断：为什么一个方案虽然好看，但不属于这个品牌。
真源已收口的素材包括：设计评审、最终取舍、风格把关、主题收口、否决方案。

一线判断
视觉上成立，但品牌上不成立
什么时候该坚持，什么时候该放掉
主题为什么会跑偏
追热点为什么会伤品牌
不能教条化的经验
不是“高级”就适合品牌
不是团队都喜欢就该保留
不是反热点就一定更高级
不是统一风格就等于品牌边界清晰
首批硬证据
一次“为什么砍掉大家都喜欢的方案”的复盘
一份“这个品牌不能长成什么样”清单
一段最能代表品牌边界的原话。
正式落点

主要进入 RoleProfile（岗位画像），作为品牌审美边界与内容专业判断的高层证据层；暂不直接硬化为规则。

7. merchandising_director（商品总监）
岗位手感

商品总监的手感在于：把好看、能卖、可补、可推、适合门店，这几件事同时放在一张图上看。
这个岗位不只看款，还看深浅、结构、上新节奏、动销、滞销、门店反馈。
真源已收口的素材包括：上新节奏安排、货品结构判断、爆款 / 平销 / 压货分析、搭配组合建议、商品和门店反馈联动。

一线判断
什么款该做深，什么款只能试水
为什么设计不错但经营上不成立
什么货适合门店，什么货适合线上
什么时候为门店让步，什么时候坚持商品逻辑
商品相关岗位最在意的搭配误区
搭得好看但卖不动
只顾单套出片，不顾货品结构
只顾模特效果，不顾普通顾客体感
只顾内容传播，不顾门店承接
只顾主推单品，不顾连带与价格带
首批硬证据
一次“设计很好但商品上不成立”的完整案例
一份“什么货不能做深”的判断清单
一段“门店反馈如何改变商品决策”的真实复盘。
正式落点

进入 RoleProfile（岗位画像）与商品经营向 TrainingMaterial（培训材料）；其中与搭配结构直接相关的部分，可作为 StylingRule（搭配规则）的经营侧证据。

六、对你提出的 5 个必答问题，正式回答
1. 老店长一眼先看什么？

先看今天这家店的节奏。
具体看货场主次、店员站位、顾客停顿点、后场能否接力；不是先看哪件衣服最漂亮。依据是店长素材包已正式要求收“顾客更像来确认还是来逛”“今天状态不对最早从哪里看出来”“高峰期先稳团队还是先冲销量”这些判断句。

2. 商品相关岗位最在意什么搭配误区？

最在意“搭得成立，但卖得不成立”。
也就是：脱离货品结构、脱离价位、脱离连带、脱离普通顾客体感、脱离门店承接能力。依据来自 merchandising_director（商品总监）和 designer（设计师）两岗的判断句与失败案例。

3. 仓库或后场岗位最怕前场怎么乱推？

最怕前场把库存事实说成销售承诺。
尤其是断码当齐码卖、替代款当同款卖、调货周期没确认就答应顾客、成套承诺和后场真实货况脱节。依据来自 warehouse_supervisor（仓库主管）的判断句与失败案例。

4. 哪些判断新人最容易学歪？

最容易学歪的是这 5 类：

把专业学成背参数
把热情学成逼单
把判断学成猜顾客
把审美学成规则
把培训学成标准答案。

这和岗位素材清单里“必须主动收失败 / 犹豫 / 卡住”和“不能只收岗位应该说什么”的正式原则完全一致。

5. 哪些经验只能作为参考，不能写成硬规则？

凡是高度依赖具体顾客、具体时段、具体门店、具体库存、具体品牌边界的判断，都只能先进 RoleProfile（岗位画像）或 TrainingMaterial（培训材料），不能直接写成 StylingRule（搭配规则）。
比如：

顾客一说“再看看”就判定没戏
某种话术永远有效
某款只要好看就该做深
某类顾客都该先推某套
仓库有货就一定能卖
