6. 正式版内容
6.1 首批必须纳入的 12 条规则
规则 01

proportion_split_rule（三七比例法）

rule_level（规则级别）：hard_rule（硬规则）
rule_content（规则内容）：全身视觉分割优先做 3:7、4:6，避免 5:5。
why_it_works（为什么成立）：衣身被正中切开时，最容易显短、显钝、显没精神；三分法穿搭本质上就是通过腰线或长度切分，让整体更利落。公开时尚指南近年仍把这条当作最基础、最好用的比例修正法之一。
suitable_for（适用于谁）：想显高、显利落、显腿长的绝大多数顾客。
not_suitable_for（不适用于谁）：明确要做低腰复古、Y2K、街头下坠感的人群。
works_in（成立场景）：高腰裤 / 高腰裙、短上衣、塞衣角、短外套。
fails_in（翻车场景）：上衣卡胯、裙长停在小腿最粗处、裤长和鞋口双截断。
substitution_boundary（替代边界）：高腰下装断码时，可换高腰同长度下装；若只能换低腰，不视作等效替代。
display_use（陈列用途）：主推人台先做“上短下长”“内短外长”。
training_use（培训用途）：试衣间纠错第一条。
规则 02

silhouette_balance_rule（松紧对冲法）

rule_level（规则级别）：hard_rule（硬规则）
rule_content（规则内容）：一身最多只留一个量感重点，上宽下收，或上收下宽。
why_it_works（为什么成立）：真实门店成交优先的是“穿上顺眼”，不是时装片效果；上下同时膨，大多数普通客群会先觉得自己变壮。
suitable_for（适用于谁）：通勤客、大众客群、第一次试穿推荐。
not_suitable_for（不适用于谁）：高个偏瘦、明确追求 oversize 街头风的顾客。
works_in（成立场景）：宽针织配直筒裤，廓形西装配窄裙，阔腿裤配合体上衣。
fails_in（翻车场景）：宽卫衣 + 宽裤 + 厚底鞋。
substitution_boundary（替代边界）：替代时先守住廓形角色，不先守颜色。
display_use（陈列用途）：整组挂场必须有量感对比。
training_use（培训用途）：教员工识别“顾客为什么觉得自己穿大了”。
规则 03

waistline_anchor_rule（腰线锚点法）

rule_level（规则级别）：hard_rule（硬规则）
rule_content（规则内容）：只要顾客诉求是显高、显瘦、显精神，先找腰线。
why_it_works（为什么成立）：门店里大部分“不够精神”不是颜色错，是腰线没出来。
suitable_for（适用于谁）：梨形、H 型、小个子、产后体型、通勤客。
not_suitable_for（不适用于谁）：腹部敏感、完全不想强调腰部的顾客。
works_in（成立场景）：高腰下装、短外套、半塞衣角、腰带辅助。
fails_in（翻车场景）：长上衣全盖、低腰下装、宽松直筒裙无收束。
substitution_boundary（替代边界）：高腰断码时，可用腰带 / 短外套 / 内外层次制造假腰线；不能直接用无腰线替代。
display_use（陈列用途）：显高显瘦主题位必用。
training_use（培训用途）：试衣镜前最容易讲明白的一条。
规则 04

one_focus_rule（一身一个主角法）

rule_level（规则级别）：hard_rule（硬规则）
rule_content（规则内容）：一套里只留一个主角：颜色主角、版型主角、材质主角、设计主角，四选一。
why_it_works（为什么成立）：零售穿搭不是造型比赛；哪里都厉害，最后往往哪里都不好卖。
suitable_for（适用于谁）：绝大多数顾客，尤其第一次种草和连带销售场景。
not_suitable_for（不适用于谁）：舞台造型、时装表达、强风格内容片。
works_in（成立场景）：亮色外套 + 基础内搭；设计感上衣 + 干净下装。
fails_in（翻车场景）：亮片上衣 + 花裙 + 金属鞋 + 大耳环。
substitution_boundary（替代边界）：主角断货时，替代保留“主角功能”，其他单品继续降噪。
display_use（陈列用途）：橱窗和人台都要有明确主角。
training_use（培训用途）：教导购别把顾客往“穿得太满”上带。
规则 05

color_echo_rule（颜色呼应法）

rule_level（规则级别）：hard_rule（硬规则）
rule_content（规则内容）：一身颜色至少有两处呼应，不让某个颜色孤零零悬着。
why_it_works（为什么成立）：呼应会显完整，也更利于做陈列分组。公开零售陈列指南也把颜色分组和焦点组织视为最直接的视觉销售工具。
suitable_for（适用于谁）：全部客群。
not_suitable_for（不适用于谁）：故意做冲突感、前卫感的拍摄型造型。
works_in（成立场景）：鞋包呼应、上衣配饰呼应、内搭下装轻呼应。
fails_in（翻车场景）：全身只有一处亮色，其他完全断开。
substitution_boundary（替代边界）：原本同色替代时，可在同温度 / 同明度内微调；不能冷暖乱跳。
display_use（陈列用途）：挂场按色阶、按温度、按系列走。
training_use（培训用途）：教新员工理解“为什么有些搭配看着散”。
规则 06

neutral_plus_accent_rule（中性色打底法）

rule_level（规则级别）：soft_rule（软建议）
rule_content（规则内容）：顾客怕出错、第一次成交、主推通勤时，先用中性色打底，再加一处提气。
why_it_works（为什么成立）：出错率低、复制率高、门店最容易稳定成交。
suitable_for（适用于谁）：通勤客、成熟客、初次尝试新色顾客。
not_suitable_for（不适用于谁）：明确要高饱和、多巴胺、节庆感的人群。
works_in（成立场景）：米白 / 灰 / 驼 / 藏青打底，丝巾 / 鞋 / 包 / 内搭提气。
fails_in（翻车场景）：全身都灰、都米，但没有材质层次。
substitution_boundary（替代边界）：提气色断货，可把提气点转移到配饰，不要硬改主上装。
display_use（陈列用途）：适合做安全成交墙。
training_use（培训用途）：给新导购的第一套安全盘。
规则 07

structure_drape_rule（挺垂对比法）

rule_level（规则级别）：soft_rule（软建议）
rule_content（规则内容）：挺的配垂的，厚的配薄的，糯的配利落的。
why_it_works（为什么成立）：材质关系决定一套衣服的“质感”和“贵气感”，不只是谁和谁同色。
suitable_for（适用于谁）：想提升质感、客单较高、贵价感商品较多的门店。
not_suitable_for（不适用于谁）：纯功能运动场景。
works_in（成立场景）：挺括西装配垂感裙，毛绒外套配牛仔，真丝衬衫配西裤。
fails_in（翻车场景）：全身都软塌，或全身都硬挺。
substitution_boundary（替代边界）：替代先守材质角色，再守颜色。
display_use（陈列用途）：相邻陈列最好有材质起伏。
training_use（培训用途）：让员工学会解释“为什么这套显贵”。
规则 08

premium_item_neutralization_rule（贵气单品中和法）

rule_level（规则级别）：soft_rule（软建议）
rule_content（规则内容）：皮草、亮片、绸缎、重工外套等贵气单品，要配基础休闲件降压。
why_it_works（为什么成立）：仓库已直接把这条作为规则样例写进抽取协议，原理是“用基础款休闲单品搭配奢华单品，降低视觉压迫感”。
suitable_for（适用于谁）：设计感强、贵价感强、容易“看着好看但顾客不敢买”的商品。
not_suitable_for（不适用于谁）：婚礼、晚宴、典礼等明确礼服场景。
works_in（成立场景）：皮草 + 牛仔，缎面半裙 + 针织，亮片上衣 + 西裤。
fails_in（翻车场景）：贵气单品再叠贵气单品。
substitution_boundary（替代边界）：中和单品必须保留“哑光、基础、低压力”特征。
display_use（陈列用途）：难卖主推款必须和基础款一起出。
training_use（培训用途）：教员工怎么把“好看但不敢买”拉回成交区。
规则 09

face_brightness_rule（近脸提气法）

rule_level（规则级别）：soft_rule（软建议）
rule_content（规则内容）：离脸最近那一层，优先解决气色，不优先解决时髦。
why_it_works（为什么成立）：顾客最直观的反馈首先是“脸色好不好”“我显不显老”。
suitable_for（适用于谁）：成熟客、黄调肤色、熬夜脸、第一次试色顾客。
not_suitable_for（不适用于谁）：强妆造、拍摄型造型。
works_in（成立场景）：围巾、内搭、领口、耳饰提亮。
fails_in（翻车场景）：整体比例都对，但近脸颜色发灰发黄。
substitution_boundary（替代边界）：上装缺货时，可优先替换近脸提亮件，不必整套推翻。
display_use（陈列用途）：近脸单品放在试镜区附近。
training_use（培训用途）：训练“先看脸，再看身”。
规则 10

formality_consistency_rule（正式度统一法）

rule_level（规则级别）：hard_rule（硬规则）
rule_content（规则内容）：正式场合下，单品正式度尽量统一。
why_it_works（为什么成立）：场合错位是最容易让顾客说不出哪里怪、但就是不买的原因。
suitable_for（适用于谁）：面试、会议、通勤、婚礼宾客、商务男装。
not_suitable_for（不适用于谁）：明确要混搭风、创意行业休闲感。
works_in（成立场景）：西装 + 皮鞋 / 乐福鞋，衬衫 + 西裤，结构连衣裙 + 干净外套。
fails_in（翻车场景）：套装配过于休闲鞋，礼服感上衣配破洞牛仔。
substitution_boundary（替代边界）：正式裤断码，只能换同级正式度单品。
display_use（陈列用途）：商务区、婚礼区、周末休闲区分开。
training_use（培训用途）：教员工识别“为什么这套不高级，是因为场合错位”。
规则 11

menswear_belt_shoe_rule（男装鞋腰带同色法）

rule_level（规则级别）：hard_rule（硬规则）
rule_content（规则内容）：男装正式穿着里，腰带与鞋的颜色、皮质感、正式度要尽量一致。
why_it_works（为什么成立）：这条在正式男装里仍是主流基础动作；公开男装风格指南也都把“鞋与腰带匹配”视为正式穿搭基本功，尤其在 suit / dress 场景下更严格。
suitable_for（适用于谁）：男装西装、商务、婚礼、面试。
not_suitable_for（不适用于谁）：纯休闲、无腰带、球鞋工装场景。
works_in（成立场景）：黑鞋黑带、棕鞋棕带、光泽与正式度相近。
fails_in（翻车场景）：黑皮鞋配浅棕编织带；正装鞋配明显牛仔休闲带。
substitution_boundary（替代边界）：鞋没货，优先找同色同正式度鞋；不是先换整套西装。
display_use（陈列用途）：男装人台必须鞋、腰带、包一起配完整。
training_use（培训用途）：补仓库里明确缺的男装规则。
规则 12

single_problem_solving_rule（单一问题优先法）

rule_level（规则级别）：hard_rule（硬规则）
rule_content（规则内容）：接客时一次先解决顾客一个核心顾虑，不同时解决五个问题。
why_it_works（为什么成立）：真正成交不是“卖点堆满”，而是“顾虑被减掉”；这条规则天然适合转成培训内容和推荐逻辑。
suitable_for（适用于谁）：全部导购、全部门店。
not_suitable_for（不适用于谁）：无。
works_in（成立场景）：顾客说显胖，就先解决显胖；说难搭，就先解决难搭。
fails_in（翻车场景）：上来就说“显高显瘦显气质显贵显年轻”。
substitution_boundary（替代边界）：原款缺货时，替代必须优先守住顾客最在意的那一个点。
display_use（陈列用途）：显高组、显瘦组、通勤组、贵价救场组分墙。
training_use（培训用途）：最适合做情境题。

6.4 哪些人群 / 身材 / 场景下规则会反转
会反转的规则一：silhouette_balance_rule（松紧对冲法）

高个、偏瘦、年轻、街头风、明确追求 oversize 时，可以上宽下宽；但它不适合作为门店安全盘默认推荐。

会反转的规则二：proportion_split_rule（三七比例法）

低腰复古、Y2K、模特感拍摄可以故意做下坠比例；但普通零售客群不能把它当主流推荐。

会反转的规则三：neutral_plus_accent_rule（中性色打底法）

节庆、活动、潮流年轻客群允许高饱和、多彩直接上身；但要先判断顾客承受力。

会反转的规则四：premium_item_neutralization_rule（贵气单品中和法）

晚宴、婚礼、典礼场景里，不一定要中和，反而可以顺着贵气走。

会反转的规则五：menswear_belt_shoe_rule（男装鞋腰带同色法）

在休闲、球鞋、工装、无腰带场景可放松；只要进入 formal dressing（正式着装），这条重新生效。

6.5 断码、换色、换材质时的替代边界

正式冻结四级优先顺序：

第一优先：守 silhouette（廓形）

原来是高腰直筒，就先找高腰直筒；原来是短外套，就先找短外套。

第二优先：守 formality_level（正式度）

商务通勤盘，不能用明显家居 / 运动单品代替。

第三优先：守 visual_weight（视觉重量）

厚薄、挺垂、毛感、光泽尽量接近。
尤其贵价单品救场时，材质像不像，往往比颜色像不像更关键。

第四优先：守 color_family（色系家族）

颜色可在同温度、同明度内微调；不能冷暖乱跳。

明确判定“不能替代”的情况
廓形变了
正式度掉了
近脸颜色把气色打脏了
材质从“贵感”换成“廉价反光”
量感从“修饰”变成“放大问题”

一句正式口径：

断码替代，不找“长得像”的，找“功能一样”的。

6.6 哪些搭配“看着成立，实际不好卖”

正式冻结为首批反例库：

all_oversized_look（全身 oversize 盘）
拍照有风格，门店里大多数顾客会觉得自己变壮。
flat_neutral_look（无层次中性色盘）
Lookbook 好看，试衣间容易被说“没精神”。
double_statement_look（上下都强设计盘）
内容里时髦，零售里容易“用力过猛”。
fragmented_leg_line_look（腿线碎裂盘）
低腰 + 裤长 / 裙长截断 + 脚口再截断，最容易显短。
luxury_overstack_look（贵气堆叠盘）
皮草配缎裙配闪鞋配大耳环，远看成立，近看日常没法卖。
formal_casual_mismatch_look（正式休闲错位盘）
上半身很正式，下半身 / 配件明显太休闲，顾客嘴上说不出，心里会退。
theme_without_problem_solution_display（只讲风格、不讲问题解决的陈列盘）
